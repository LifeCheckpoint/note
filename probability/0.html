两点分布 (Bernoulli 分布) `p_1 = p`, `p_0 = q` `0 lt p lt 1`, `p + q = 1` `p` `pq` `p"e"^(it) + q`
二项分布 `B(n, p)` `b(k; n, p) = ((n),(k)) p^k q^(n-k)` `k in [0, n] nn ZZ`, `0 lt p lt 1`, `p + q = 1` `np` `npq` `(p"e"^(it) + q)^n`
Poisson 分布 `P(lambda)` `p(k;lambda) = lambda^k/(k!) "e"^-lambda` `k = {0} uu ZZ^+`, `lambda gt 0` `lambda` `lambda` `"e"^(lambda("e"^(it)-1))`
几何分布 `g(k;p) = q^(k-1) p` `k in ZZ^+`, `0 lt p lt 1`, `p + q = 1` `1/p` `q/p^2` `(p"e"^(it))/(1-q"e"^(it))`
超几何分布 `p_k = [((M),(k)) ((N-M),(n-k))]/{:((N),(n)):}` `M, N, n in ZZ^+`, `M, n le N`, `k = [0, min(M, n)] nn ZZ` `(nM)/N` `(nM)/N (1-M/N) * (N-n)/(N-1)` `sum_(k=0)^n p_k "e"^(itk)`
Pascal 分布 `p_k = ((k-1), (r-1)) p^r q^(k-r)` `r in ZZ^+`, `k in [r, oo) nn ZZ`, `0 lt p lt 1`, `p + q = 1` `r/p` `(rq)/p^2` `( (p"e"^(it)) / (1-q"e"^(it)) )^r`
负二项分布 `p_k = ((-r),(k)) p^r (-q)^k` `k in {0} nn ZZ^+`, `0 lt p lt 1`, `p + q = 1`, `r gt 0` `(rq)/p` `(rq)/p^2` `(p/(1-q"e"^(it)))^r`

常用连续型分布

名称 概率密度函数 `p(x)` 备注 数学期望 `int_-oo^oo x "d"F(x)` 方差 `E(xi-E xi)^2` 特征函数 `E "e"^(it xi)`
正态分布 (Gauss 分布) `N(mu, sigma^2)` `1/(sqrt(2 pi) sigma) "e"^(-(x-mu)^2/(2sigma^2))` `x in RR`, `mu, sigma gt 0` 为常数 `mu` `sigma^2` `exp(i mu t - 1/2 sigma^2 t^2)`
均匀分布 `U[a, b]` `{(1/(b-a), if a le x le b),(0, if "else"):}` `-oo lt a lt b lt oo` 为常数 `(a+b)/2` `(b-a)^2/12` `("e"^(itb)-"e"^(ita))/(it(b-a))`
指数分布 `"Exp"(lambda) = Gamma(1, lambda)` `{(lambda "e"^(-lambda x), if x ge 0),(0, if "else"):}` `lambda gt 0` 为常数 `lambda^-1 `lambda^-2 `(1-(it)/lambda)^-1`
`chi^2` 分布 `= Gamma(n/2, 1/2)` `{((x^(n/2-1) "e"^(-x/2))/(2^(n/2) Gamma(n/2)), if x ge 0), (0, if "else"):}` `n in ZZ^+` `n` `2n` `(1-2it)^(-n/2)`
`Gamma` 分布 `Gamma(r, lambda)` `{( lambda^r/(Gamma(r)) x^(r-1) "e"^(-lambda x), if x ge 0), (0, if "else"):}` `r, lambda gt 0` 为常数 `r lambda^-1` `r lambda^-2` `(1-(it)/lambda)^-r`
Cauchy 分布 `1/pi * lambda/(lambda^2 + (x-mu)^2)` `x in RR`; `lambda gt 0` 为常数, `mu` 为常数 不存在 `exp(i mu t - lambda|t|)`
t 分布 `(Gamma((n+1)/2))/(sqrt(n pi) Gamma(n/2)) (1+x^2/n)^(-(n+1)/2)` `x in RR`, `n in ZZ^+` `0` (`n gt 1`) `n/(n-2)` (`n gt 2`)
Pareto 分布 `{((r A^r)/x^(r+1), if x ge A), (0, if "else"):}` `r, A gt 0` `r gt 1` 时存在 `r gt 2` 时存在
F 分布 `{( (m^(m/2) n^(n/2))/(B(m/2, n/2)) * x^(m/2-1)/(n+mx)^((m+n)/2), if x ge 0), (0, if "else"):}` `m, n in ZZ^+` `n/(n-2)` (`n gt 2`) `(2n^2(m+n-2))/(m(n-2)^2(n-4))` (`n gt 4`)
Beta 分布 `{((x^(p-1) (1-x)^(q-1))/(B(p, q)), if 0 lt x lt 1), (0, if "else"):}` `p, q gt 0` 为常数 `p/(p+q)` `(pq)/((p+q)^2 (p+q+1))` `sum_(k=0)^oo (B(p+q, p+k) (it)^k)/(B(p, p+q+k) k!)`
对数正态分布 `{(1/(sigma x sqrt(2 pi)) "e"^(-(ln x-alpha)^2/(2 sigma^2)), if x gt 0), (0, if "else"):} `alpha, sigma gt 0` 为常数 `"e"^(alpha + sigma^2/2)` `"e"^(2 alpha + sigma^2) ("e"^(sigma^2) - 1)`
Weibull 分布 `{(alpha lambda x^(alpha-1) "e"^(-lambda x^alpha), if x gt 0), (0, if "else"):}` `alpha, lambda gt 0` 为常数 `(Gamma(1/alpha + 1))/lambda^(1/alpha)` `[Gamma(2/alpha+1) - (Gamma(1/alpha+1))^2]/lambda^(2/alpha)`
