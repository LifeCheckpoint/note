类比于平面仿射几何系统, 我们抽象出线性空间的概念:

    令集合 `V != O/`, `bbb P` 为一数域, "`+`" 为 `v` 上的一个二元运算, 称为加法, "`*`" 为 `bbb P xx V` 到 `V` 的一个二元合成, 称为数乘. (注意, "二元运算" 与 "二元合成" 隐含 `V` 对运算封闭, 即对任意 `bm alpha, bm beta in V` 和任意 `k in bbb P`, `bm alpha + bm beta in V`, `quad` `k bm alpha in V`. 称代数系统 `(V; +, *; bbb P)` 为数域 `bbb P` 上的线性空间, 如果它满足下面 8 条基本性质:
  1. 加法交换律. `(AA bm alpha, bm beta in V)` `bm alpha + bm beta = bm beta + bm alpha`;
  2. 加法结合律. `(AA bm alpha, bm beta, bm gamma in V)` `(bm alpha + bm beta) + bm gamma = bm alpha + (bm beta + bm gamma)`;
  3. (右) 零元存在. `(EE bm theta in V)` `(AA bm alpha in V)` `bm alpha + bm theta = bm alpha`;
  4. (右) 负元存在. `(AA bm alpha in V)` `(EE bm alpha' in V)` `bm alpha + bm alpha' = bm theta`;
  5. 数乘对 `V` 上加法的分配律. `(AA bm alpha, bm beta in V & AA k in bbb P)` `k * (bm alpha + bm beta) = k * bm alpha + k * bm beta`;
  6. 数乘对 `bbb P` 上加法的分配律. `(AA bm alpha in V & AA k, l in bbb P)` `(k + l) * bm alpha = k * bm alpha + l * bm alpha`;
  7. 数乘下数字的穿越. `(AA bm alpha in V & AA k, l in bbb P)` `(kl) * bm alpha = k * (l * bm alpha)`;
  8. 设 `1` 是 `bbb P` 中的幺元, `(AA bm alpha in V)` `1 * bm alpha = bm alpha`.
    9. 不引起混淆的情况下, 把线性空间也简记简记为 `V`. 把线性空间中的元素称为向量, 数乘的乘号可以省略.

线性空间的基本性质 i - iv 指出 `(V, +)` 构成一个 Abel 群. 若把数域 `bbb P` 换成一般的域, 这里定义的就是抽象代数中的模.

    线性空间的简单性质
  1. iii 中的 `bm theta` 是唯一的, 称为线性空间 `V` 中的零向量;
  2. 设 `0` 是 `bbb P` 中的零元, `(AA bm alpha in V)` `0 bm alpha = bm theta`.
  3. `(AA k in bbb P)` `k bm theta = bm theta`;
  4. 对任意 `bm alpha in V`, iv 中的 `bm alpha'` 是唯一的, 称为 `bm alpha` 的负向量, 记为 `-bm alpha`; 因此, `-(-bm alpha) = bm alpha`;
  5. `(AA bm alpha in V)` `(-1) bm alpha = -bm alpha`;
  6. 消去律. `(AA bm alpha, beta, gamma in V)` `bm alpha + bm beta = bm alpha + bm gamma iff bm beta = bm gamma`.
  7. `(AA bm alpha in V & AA k in bbb P)` `k bm alpha = bm theta rArr k = 0 or bm alpha = bm theta`;
  1. 若 `bm theta'` 也是 `V` 的零向量, 则由交换律 (i) 和零元的定义 (iii) 有 ` bm theta' = bm theta' + bm theta = bm theta + bm theta' = bm theta`.
  2. 由 viii 和 vi 有 ` bm alpha + 0 bm alpha = 1 bm alpha + 0 bm alpha = (1 + 0) bm alpha = 1 bm alpha = bm alpha`. 由零元唯一性知 `0 bm alpha = bm theta`.
  3. 任取 `bm alpha in V`, 由简单性质 2 和 vii , ` k bm theta = k (0 bm alpha) = (k*0) bm alpha = 0 bm alpha = bm theta`.
  4. 设 `bm alpha''` 也是 `bm alpha` 的负向量, 则由交换律, 结合律和零元, 负元的定义有 ` bm alpha'' = bm alpha'' + bm theta = bm theta + bm alpha'' = bm alpha + bm alpha' + bm alpha'' = bm alpha' + bm alpha + bm alpha'' = bm alpha' + bm theta = bm alpha'`.
  5. 由简单性质 2, ` bm alpha + (-1) bm alpha = 1 bm alpha + (-1) bm alpha = (1-1) bm alpha = 0 bm alpha = bm theta`. 由负元唯一性知 `(-1) bm alpha = -bm alpha`.
  6. `lArr`: 两边同时加上 `bm alpha`; `rArr`: 两边同时加上 `-bm alpha`.
  7. 设 `k != 0`, 则由简单性质 3, ` bm alpha = 1 bm alpha = (k^-1 k) bm alpha = k^-1 (k bm alpha) = k^-1 bm theta = bm theta`.
    零元, 负元的唯一性和消去律都是群的性质, 我们只由 ii, iii, iv 就可以推出它们:
  1. 右负元也是左负元: 对 `AA bm alpha in V`, 有 `bm alpha', (bm alpha')' in V`, 满足 ` bm alpha + bm alpha' = bm alpha' + (bm alpha')' = bm theta`. 于是 ` bm alpha' + bm alpha = bm alpha' + bm alpha + bm theta = bm alpha' + bm alpha + bm alpha' + (bm alpha')'` `= bm alpha' + bm theta + (bm alpha')' = bm alpha' + (bm alpha')' = bm theta`.
  2. 右零元也是左零元: 对 `AA bm alpha in V` ` bm theta + bm alpha = bm alpha + bm alpha' + bm alpha = bm alpha + bm theta = bm alpha`.
    3. 因此当 `bm alpha, bm beta in V` 中有一个是零元, 或者有一个是另一个的负元时, `bm alpha + bm beta = bm beta + bm alpha`. 从而上述推论 1, 4, 6 的证明实际上不依赖于交换律.

线性空间定义中的 8 条基本性质不是相互独立的, 比如我们可以用 ii, iii, iv, v, vi, viii 来推出 i. 对任意 `bm gamma in V`, 由 vi 和 viii, ` 2 bm gamma = (1+1) bm gamma = 1 bm gamma + 1 bm gamma = bm gamma + bm gamma`. 于是对任意 `bm alpha, bm beta in V`, 由 ii, ` 2 (bm alpha + bm beta) = (bm alpha + bm beta) + (bm alpha + bm beta) = bm alpha + (bm beta + bm alpha) + bm beta`. 但由 v, ii 知 ` 2 (bm alpha + bm beta) = 2 bm alpha + 2 bm beta = (bm alpha + bm alpha) + (bm beta + bm beta) = bm alpha + (bm alpha + bm beta) + bm beta`. 由消去律 (从注4-2 知道, 消去律只依赖于 ii, iii, iv) 知 `bm beta + bm alpha = bm alpha + bm beta`.

令 `(V; +, *; bbb P)` 为一线性空间, `O/ != V_1 sube V`. 若 `(V_1; +, *, bbb P)` 也为一线性空间, 则称 `V_1` 为 `V` 的线性子空间, 简称子空间, 记为 `V_1 le V`. 进一步, 若 `V_1 != V`, 称 `V_1` 为 `V` 的真子空间, 记为 `V_1 lt V`. 显然对于任意数域 `bbb P` 上的任意线性空间 `V`, `{bm theta}` 和 `V` 本身都是 `V` 的子空间, 称为 `V` 的平凡子空间.

    令 `(V; +, *; bbb P)` 为一线性空间, `V_1 sube V`. 则 `V_1` 为 `V` 的子空间的充要条件是
  1. `V_1 != O/`;
  2. `V_1` 对 `V` 上的加法与数乘封闭.
    3. 从而我们在验证线性子空间时, 不需要烦琐地逐一验证 8 条性质.

必要性显然, 今证充分性, 这只需再证明 `V_1` 满足定义4-1 的 8 条性质. 由于 `V_1 sube V`, i, ii 及 v - viii 显然成立. 接下来证明 iii, iv 成立. 任取 `bm alpha in V_1`, 则由推论4-1 的 2 及 `V_1` 对数乘的封闭性, `bm theta = 0 bm alpha in V_1`, 故 iii 成立. 同理, 对任意 `bm alpha in V_1`, `-bm alpha = (-1) bm alpha in V_1`, 故 iv 成立.

令 `V` 为 `bbb P` 上一线性空间, `bm alpha_1, bm alpha_2, cdots, bm alpha_n in V`, 易知 `{sum_(i=1)^n k_i bm alpha_i: k_i in bbb P, i = 1, 2, cdots, n} le V`, 称为由 `bm alpha_1, bm alpha_2, cdots, bm alpha_n` 生成的子空间, 记为 `G[bm alpha_1, bm alpha_2, cdots, bm alpha_n]`.
