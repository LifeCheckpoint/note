矩阵的定义及其运算

矩阵的线性运算, 矩阵的乘法

一个记号. 设 `n` 为正整数, 记集合 `[n] = {1, 2, cdots, n}`.

令 `bbb P` 为一数域, `bbb P` 上 `m` 行 `n` 列矩阵的全体记为 `bbb P^(m xx n)`, `bbb P` 称为矩阵的基础数域. 如果 `m xx n` 矩阵 `bm A = (a_(i j))` 和 `bm B = (b_(i j))` 满足 `a_(i j) = b_(i j)`, `quad i in [m], j in [n]`, 则称它们相等, 记为 `bm A = bm B`.

    矩阵的线性运算 设 `c in bbb P`, `bm A = (a_(i j))_(m xx n)`, `bm B = (b_(i j))_(m xx n)`.
  1. 数乘. 定义 `c bm A := (c_(i j))_(m xx n)`, 其中 `c_(i j) = c a_(i j)`, `quad i in [m], j in [n]`. 即矩阵与 `c` 数乘, 相当于把每一元素都乘以 `c`.
  2. 加法. 定义 `bm (A+B) := (s_(i j))_(m xx n)`, 其中 `s_(i j) = a_(i j) + b_(i j)`, `quad i in [m], j in [n]`. 即两矩阵相加, 相当于把对应元素相加.

矩阵乘法 令 `bm A = (a_(i j))_(m xx n)`, `bm B = (b_(i j))_(n xx l)`. 定义 `bm (A B) := (p_(i j))_(m xx l)`, 其中 `p_(i j) = sum_(k=1)^n a_(i k) b_(k j)`, `quad i in [m], j in [l]`. 这一乘法法则称为 "左行乘右列", 即矩阵 `bm (A B)` 的 `i j` 元由 `bm A` 的第 `i` 行与 `bm B` 的第 `j` 列作向量的一般内积得到. 为何这样定义矩阵乘法呢, 考虑两个线性函数组 `y_i = sum_(k=1)^n a_(i k) x_k`, `quad i in [m]`;
`x_k = sum_(j=1)^l b_(k j) t_j`, `quad k in [n]`. 将第二式代入第一式, `y_i = sum_(k=1)^n a_(i k) sum_(j=1)^l b_(k j) t_j` `= sum_(j=1)^l t_j sum_(k=1)^n a_(i k) b_(k j)` `= sum_(j=1)^l p_(i j) t_j`. 从而矩阵乘法反映了线性函数组的复合.

  1. 元素全为零的矩阵称为零矩阵, 记为 `bm O`. 矩阵加上零矩阵, 结果不变. 矩阵左乘或右乘零矩阵, 结果为零矩阵.
  2. 矩阵 `bm A` 与 `-1` 数乘的结果记为 `-bm A`. 显然 `bm A + (-bm A) = bm O`. 从而矩阵的减法定义为 `bm (A-B) := bm A + (-bm B)`.
  3. `n xx n` 矩阵称为 `n` 阶方阵; 矩阵的 `i = j` 的 `i j` 元组成它的主对角线: `{a_(i j) | i = j}`. `n` 阶方阵的 `i + j = n+1` 的 `i j` 元组成它的副对角线: `{a_(i j) | i + j = n}`.
  4. 如果方阵的所有满足 `i lt j` 的 `i j` 元均为零, 则称它为 `n` 阶下三角矩阵; 如果方阵的所有满足 `i le j` 的 `i j` 元均为零, 则称它为 `n` 阶严格下三角矩阵; 如果方阵的所有满足 `i gt j` 的 `i j` 元均为零, 则称它为 `n` 阶上三角矩阵; 如果方阵的所有满足 `i ge j` 的 `i j` 元均为零, 则称它为 `n` 阶严格上三角矩阵;
  5. 如果方阵的主对角线以外的元素均等于零, 即矩阵形如 `[ (d_1" ",0" ",cdots" ",0), (0" ",d_2" ",cdots" ",0), (vdots" ",vdots" ", " ",vdots), (0" ",0" ",cdots" ",d_n) ]`, 则称它为 `n` 阶对角矩阵, 记为 `bm D_(d_k)`. 对角矩阵既是上三角矩阵, 也是下三角矩阵. 矩阵左乘 `bm D_(d_k)`, 相当于第 `i` 行每个元素都乘以 `d_i`; 矩阵右乘 `bm D_(d_k)`, 相当于第 `j` 列每个元素都乘以 `d_j`. 对角矩阵相乘, 结果还是对角矩阵.
  6. 如果 `n` 阶对角阵的每个对角元都等于 `1`, 则称它为 `n` 阶单位矩阵, 记为 `bm E_n` 或 `bm I_n`, 或者省略下标, 简记为 `bm E` 或 `bm I`. 如果引入 Kronecker `delta` 函数 `delta_(i j) = { (1, if i = j;), (0, if i != j;) :}` 则单位阵的 `i j` 元就是 `delta_(i j)`. 矩阵左乘或右乘单位矩阵, 结果都不变.
  7. 如果 `n` 阶对角阵的每个对角元都等于常数 `c`, 即矩阵等于 `c bm E`, 则称它为 `n` 阶数量矩阵或纯量矩阵. 矩阵左乘或右乘 `c bm E`, 结果相当于与 `c` 作数乘.
    假设下面各式中的加法与乘法都可以进行, 即矩阵具有合适的行, 列数.
  1. 加法交换律. `bm (A + B) = bm (B + A)`.
  2. 加法结合律. `bm ((A+B)+C) = bm (A+(B+C))`.
  3. 乘法结合律. `bm ((A B)C) = bm (A(B C))`.
  4. 乘法对加法的分配律. `bm (A(B+C)) = bm (A B + A C)`, `bm ((B+C)A) = bm (B A + C A)`.
  5. 数乘的穿越. `c(bm(A B)) = (c bm A)bm B = bm A(c bm B)`.
  6. 数乘的结合律. `(c d)bm A = c(d bm A)`.
  7. 数乘的单位元. `1 bm A = bm A`.
  8. 数乘的分配律. `c(bm A + bm B) = c bm A + c bm B`, `(c + d)(bm A) = c bm A + d bm A`.

矩阵乘法一般不成立交换律, 如 `bm A, bm B` 分别是 `m xx n`, `n xx m` 矩阵, 则 `bm (A B)` 是 `m` 阶方阵, 而 `bm (B A)` 是 `n` 阶方阵, 因此两个矩阵的乘法可交换的必要条件是 `m = n`. 即使有 `m = n`, `bm (A B) = bm (B A)` 也未必成立. 令 `bm A = [(1" ",0),(0" ",0)]`, `quad bm B = [(0" ",1),(0" ",0)]`, 则 `bm (A B) = bm B`, `bm (B A) = bm O`.
不过, 方阵与零矩阵, 方阵与数量矩阵 (特别地, 单位阵) 的乘法可以交换. 两个对角矩阵的乘法可以交换. 方阵与自身的乘法, 或者与其逆矩阵 (见下文) 的乘法可以交换.

矩阵的转置

令 `bm A = (a_(i j))_(m xx n)`, 定义 `bm A` 的转置 `bm A' := (t_(i j))_(n xx m)`, 其中 `t_(i j) = a_(j i)`, `quad i in [n], j in [m]`. 矩阵的转置即 "行变列, 列变行". 形象地说, 转置就是将矩阵沿主对角线翻转.

  1. `(bm A')' = bm A`.
  2. `(c bm A)' = c bm A'`.
  3. `(bm A + bm B)' = bm A + bm B`.
  4. `(bm (A B))' = bm B' bm A'`.
  1. 上, 下三角矩阵互为转置; 严格上, 下三角矩阵互为转置.
  2. 如果方阵 `bm A` 满足 `bm A' = bm A`, 则称它为对称矩阵; 如果方阵 `bm A` 满足 `bm A' = -bm A`, 则称它为反对称矩阵. 如同任一函数可分解为偶函数与奇函数之和: `f(x) = (f(x) + f(-x))/2 + (f(x) - f(-x))/2`, 任一方阵也可分解为对称矩阵与反对称矩阵之和: `bm A = (bm A + bm A')/2 + (bm A - bm A')/2`.
  3. 把其他位置都为零, 而第 `j` 个下标处等于 `1` 的单位列向量记作 `bm epsi_j := (overbrace(0,cdots,0,1)^j ,0,cdots,0)'`; 其他位置都为零, 而第 `i` 行 `j` 列处等于 `1` 的矩阵记作 `bm E_(i j) = (overbrace(bb 0,cdots,bb 0,bm epsi_i)^j,bb 0,cdots, bb 0)`. 分别用 `bm alpha_i`, `bm A_j` 记矩阵 `bm A` 的第 `i` 行和第 `j` 列, 有 `bm (A epsi)_j = bm A_j`, `quad bm epsi_i' bm A = bm alpha_i`,
    `bm epsi_i' bm epsi_j = delta_(i j)`, `quad bm epsi_j bm epsi_i' = bm E_(i j)`.

矩阵的逆

令 `bm A, bm B` 是 `n` 阶方阵, 如果 `bm (A B) = bm E`, 则称 `bm A` 是 `bm B` 的左逆, `bm B` 是 `bm A` 的右逆. 如果 `bm B` 同时是 `bm A` 的左逆和右逆, 即 `bm (A B) = bm (B A) = bm E`, 则称 `bm B` 是 `bm A` 的逆矩阵. 如果 `bm A` 的逆矩阵存在, 则称它是可逆的. 如果 `bm A` 可逆, 则它的逆矩阵必惟一, 记作 `bm A^-1`.

设 `bm B, bm C` 都是 `bm A` 的逆. 则 `bm B = bm (B E) = bm (B(A C))` `= bm ((B A)C) = bm (E C) = bm C`. 因此逆矩阵必惟一.

利用行列式的知识可以证明, 如果 `bm A` 的左逆存在, 则它也是 `bm A` 的右逆, 反之亦然. 因此只要 `bm (A B) = bm E`, 就可以断言 `bm A, bm B` 都可逆, 且它们互逆.
事实上, 由 `bm (A B) = bm E` 有 `|bm A||bm B| = 1`, 从而 `|bm A| != 0`, 即 `bm A` 可逆. 记 `bm A` 的逆为 `bm A^-1`, 于是 `bm B = bm (E B) = bm (A^-1 A B)` `= bm (A^-1 E) = bm A^-1`.

    设 `bm A`, `bm B` 是 `n` 阶可逆方阵. `c in bbb P`, `c != 0`, 则下面各矩阵也可逆, 它们的逆写在等号右边.
  1. `(bm A^-1)^-1 = bm A`.
  2. `(bm(A B))^-1 = bm B^-1 bm A^-1`.
  3. `(bm A')^-1 = (bm A^-1)'`.
  4. `(c bm A)^-1 = c^-1 bm A^-1`.

矩阵运算进阶

几类矩阵可交换的结论

  1. 设 `bm A, bm B` 是同阶对称矩阵, 则 `bm(A B)` 是对称矩阵当且仅当 `bm A, bm B` 可交换.
  2. 如果方阵 `bm A` 满足 `bm A^2 = bm E`, 则称它为对合矩阵. 对合矩阵的逆就是自己. 设 `bm A, bm B` 是同阶对合矩阵, 则 `bm(A B)` 是对合矩阵当且仅当 `bm A, bm B` 可交换.
  3. 如果方阵 `bm A` 满足 `bm A^2 = bm A`, 则称它为幂等矩阵. 所有幂等矩阵中只有单位阵可逆. 设 `bm A, bm B` 是同阶幂等矩阵, 则 `bm(A+B)` 是幂等矩阵当且仅当 `bm (A B) = bm (B A) = bm O`.
  1. 设 `bm((A B))' = bm(A B)`, 即 `bm((A B))' = bm((B' A'))' = bm((B A))'`, 即 `bm(A B) = bm(B A)`. 反面的证明是容易的.
  2. 先设 `bm(A B A B) = bm E`, 于是 `bm B^-1 = bm(A B A)`, `bm A^-1 = bm(B A B)`. 因为 `bm A^2 = bm B^2 = bm E`, 有 `bm(A B A B) = bm A^2 bm B^2`, 左边同乘以 `bm A^-1`, 右边同乘以 `bm B^-1` 就得到 `bm(B A) = bm(A B)`. 反之类似.
  3. 先设 `bm((A+B))^2 = bm A + bm B`, 于是 `bm A + bm B` `= bm A^2 + bm(A B + B A) + bm B^2` `= bm A + bm(A B + B A) + bm B`. 比较得 `bm(A B) = -bm(B A)`. 于是 `bm(A B) = bm(A A B)` `= bm(A (-B A)) = -bm((A B)A)` `= bm(B A A) = bm(B A)`. 但 `bm(A B) = -bm(B A)`, 于是 `bm(A B) = bm(B A) = bm O`. 反面的证明的容易的.

方阵的幂与多项式

方阵的幂 设 `bm A` 是方阵, `n` 为正整数, 则 `bm A^0 := bm E`, `quad bm A^n := bm (A A)^(n-1)`. 幂运算满足 `bm A^m bm A^n = bm A^(m+n)`, `quad (bm A^m)^n = bm A^(m n)`. 显然 `bm A^m`, `bm A^n` 是可交换的. 当 `bm A, bm B` 可交换时, `bm A^n bm B^n = (bm (A B))^n`. 设 `bm P` 可逆, 则 `(bm (P^-1 A P))^n = bm P^-1 bm A^n bm P`. 分块对角矩阵的幂 `bm D_(bm A_k)^n = bm D_(bm A_k^n)`.

方阵的多项式 令 `f(x) = sum_(k=0)^m c_k x^k`, `a_m != 0` 为一多项式, `bm A = (a_(i j))_(n xx n)`, 则 `f(bm A) = sum_(k=0)^n c_k bm A^k` 有意义, 且 `f(bm A) = (f(a_(i j)))_(n xx n)`. 于是方阵的多项式可以等同于多项式的方阵.

设 `bm A` 是方阵. 由等式 `bm E - bm A^n = bm((E-A)) sum_(k=0)^(n-1) bm A^k` 知道, 若 `bm A^n = bm O`, 则 `bm E - bm A` 可逆, `bm((E-A))^-1 = sum_(k=0)^(n-1) bm A^k`.

方阵的迹, 正交矩阵

设 `bm A = (a_(i j))_(n xx n)`, 定义 `bm A` 的迹为它的主对角线上的元素之和, 记为 `"tr"bm A := sum_(i=1)^n a_(i i)`. 方阵的迹满足 `"tr"(bm E_(n xx n)) = n`, `quad "tr"(c bm A) = c "tr"bm A`, `quad "tr"(bm(A+B)) = "tr"bm A + "tr" bm B`,
`"tr"(bm (A B)) = "tr"(bm (B A))`, `quad "tr"(bm A') = "tr"bm A`. 任意 `n` 阶方阵可以写成一个迹为零的矩阵与一个数量矩阵之和: 记 `1/n "tr" bm A = t`. `bm A = (bm A - t bm E) + t bm E`.

设 `bm A = (a_(i j))_(m xx n)`, `bm bar A = (bar(a_(i j)))_(m xx n)`, 则 `bm (bar A' A)` 是 `n` 阶对称矩阵 (注意内积的共轭对称性), 且 `"tr"(bm(bar A' A))` 等于 `bm A` 的每个元素的模的平方和, 即 `"tr"(bm(bar A'A)) = sum_(i=1)^m sum_(j=1)^n |a_(i j)|^2`. 从而 `bm A = bm O` 当且仅当 `"tr"(bm(bar A'A)) = 0`.

如果方阵 `bm T` 满足 `bm (T' T) = bm E`, 则称 `bm T` 为 `n` 阶正交矩阵 (orthgonal matrix). 正交矩阵的第 `i` 行与第 `j` 行的向量内积等于 `delta_(i j)`. 例如下面的矩阵是正交矩阵: `[(cos theta" ",-sin theta),(sin theta" ",cos theta)]`. 用这个矩阵左乘一个平面向量, 结果恰好将该向量逆时针旋转 `theta` 角.

    设 `bm A, bm B` 为方阵,
  1. 如果存在可逆矩阵 `bm T` 使得 `bm B = bm(T'A T)`, 则称矩阵 `bm A, bm B` 合同. 合同是一等价关系. 如果 `bm A` 为对称矩阵, 则任意与 `bm A` 合同的矩阵都对称. 称方阵 `bm A` 正定, 如果它合同于单位矩阵 `bm E`.
  2. 如果存在可逆矩阵 `bm T` 使得 `bm B = bm(T^-1 A T)`, 则称矩阵 `bm A, bm B` 相似. 相似是一等价关系. 特别若 `bm T` 是正交矩阵, 则称 `bm A, bm B` 正交相似. 显然正交相似的两个矩阵既合同, 又相似.

方阵逆的一些结论

分块对角矩阵的逆. 假设每个 `bm A_k` 都可逆, 则 `bm D_(bm A_k)^-1 = bm D_(bm A_k^-1)`.

设 `bm A, bm B` 可逆, 则 `[(bm O" ",bm A),(bm B" ",bm O)]^-1` `= [(bm O,bm B^-1),(bm A^-1,bm O)]`.

设 `bm A, bm B` 是 `n` 阶方阵, `bm A, bm B, bm A + bm B` 都可逆, 则 `bm A^-1 + bm B^-1 = bm B^-1(bm B+bm A)bm A^-1` `= bm A^-1(bm B+bm A)bm B^-1`. 容易看出 `bm A^-1 + bm B^-1` 也可逆.

设 `bm A, bm B` 是 `n` 阶方阵, 且 `bm (A+B) = bm (A B)`, 则 `bm A, bm B` 可交换.

因为 `bm((E-A)(E-B))` `= bm E - bm((A+B)) + bm(A B)` `= bm E`, 所以 `bm(E-A), bm(E-B)` 互逆. 得到 `bm E = bm((E-B)(E-A))` `= bm E - bm((A+B)) + bm(B A)`. 从而 `bm (B A) = bm(A+B) = bm(A B)`.

设 `bm E + bm(A B)` 可逆, 证明 `bm E + bm(B A)` 也可逆.

设 `bm X = (bm E + bm(A B))^-1`, 则 `bm(B X A)(bm E + bm(B A))` `= bm(B X)(bm E + bm(A B))bm A` `= bm(B A)`,
`(bm E + bm(B A))bm(B X A)` `= bm B(bm E + bm(A B))bm(X A)` `= bm(B A)`. 故 `bm E - bm(B X A) = (bm E + bm(B A))^-1`.

矩阵的初等变换

    规定矩阵的三种行初等变换:
  1. 将矩阵第 `i` 行的每个元素都乘以非零常数 `c`, 记作 `[i(c)]`;
  2. 将矩阵第 `j` 行的 `k` 倍加到第 `i` 行上, 记作 `[i+j(k)]`;
  3. 将矩阵第 `i`, `j` 行交换位置, 记作 `[i,j]`.
    4. 类似可规定三种列初等变换, 分别记为 `{i(c)}`, `{i+j(k)}`, `{i,j}`.
    将三种行 (列) 初等变换分别作用于 `n` 阶单位矩阵, 得到 第一类初等矩阵: `[(bm E_(i-1)" ", ,), (,c" ",), (, ,bm E_(n-i))]` `= (epsi_1, cdots, epsi_(i-1), c epsi_i, epsi_(i+1), cdots, epsi_n)`. 第二类初等矩阵: `[(bm E_(i-1)" ", , , ,), (,1" ",cdots" ",k" ",), (, ,ddots" ",vdots" ",), (, , ,1" ",), (, , , ,bm E_(n-j))]` `= (epsi_1, cdots, epsi_(j-1), k epsi_i + epsi_j, epsi_(j+1), cdots, epsi_n)`. 和第三类初等矩阵: `[(bm E_(i-1)" ", , , ,), (,0" ",cdots" ",1" ",), (,vdots" ",bm E_(j-i-1)" ",vdots" ",), (,1" ",cdots" ",0" ",), (, , , ,bm E_(n-j))]` `= (epsi_1, cdots, epsi_(i-1), epsi_j, epsi_(i+1), cdots, epsi_(j-1), epsi_i, epsi_(j+1), cdots, epsi_n)`. 容易计算验证, 用第 `i` 类初等矩阵左 (右) 乘一矩阵 `bm A`, 相当于对 `bm A` 施以第 `i` 类初等行 (列) 变换, `i = 1,2,3`. 初等矩阵都是可逆的, 它们的逆分别是同一类的初等矩阵.

如果 `m xx n` 矩阵 `bm A` 可经有限次初等变换化为 `bm B`, 则称 `bm A, bm B` 等价, 记为 `bm A ~ bm B`. 矩阵的等价是一种等价关系.

`m xx n` 阶矩阵 `bm A, bm B` 等价当且仅当存在 `m` 阶初等矩阵 `{bm F_i}_(i=1)^s` 和 `n` 阶初等矩阵 `{bm G_j}_(j=1)^t`, 使得 `bm B = bm F_s cdots bm F_2 bm F_1 bm A bm G_1 bm G_2 cdots bm G_t`.

任意 `m xx n` 矩阵等价于如下形状的矩阵 `bar bm A = [(bm E_r" ", bm O),(bm O" ", bm O)]_(m xx n)`, 称为 `bm A` 的等价标准形, 其中 `0 le r le min{m,n}`.
