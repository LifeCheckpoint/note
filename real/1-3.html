映射

设 `X, Y != O/`, 若对每个 `x in X`, 存在唯一的 `y in Y` 与之对应, 则称从 X 到 Y 建立了一个映射, 如用 `f` 表示这一映射, 可记为 `f: X |-> Y`. 称一个映射是良定义的, 如果上述的存在唯一性被满足, 即 1. 对任意 `x in X`, `f(x)` 存在且 `in Y`; 2. `x_1 = x_2 rArr f(x_1) = f(x_2)`.

设 `f` 和 `g` 都是 `X` 到 `Y` 的映射, 称 `f` 与 `g` 相等, 如果对任意 `x in X`, `f(x) = g(x)`.

若 `f` 把 `x in X` 映到 `y in Y`, (记为 `y = f(x)`), 则称 `y` 为 `x` 在 `f` 下的象, `x` 为 `y` 在 `f` 下的一个原象.

设 `f: X |-> Y`, `A sube X`, 记 `f(A) = {f(x): x in A} = {y in Y: (EE x in A) y = f(x)} sube Y` 为 `A` 在 `f` 下的象集. 约定 `f(O/) = O/`. 特别 `f(X)` 也记为 `"Im"\ f`, 称为 `f` 的象. 我们有 `A_1 sube A_2 rArr f(A_1) sube f(A_2)` (单调性)
`f(uuu_(alpha in I) A_alpha) = uuu_(alpha in I) f(A_alpha)`
`f(nnn_(alpha in I) A_alpha) sube nnn_(alpha in I) f(A_alpha)`

设 `f: X |-> Y`, `B sube Y`, 记 `f^-1(B) = {x in X: f(x) in B} sube X` 为 `B` 在 `f` 下的原象集. 我们有 `B_1 sube B_2 rArr f^-1(B_1) sube f^-1(B_2)` (单调性)
`f^-1(uuu_(alpha in I) B_alpha) = uuu_(alpha in I) f^-1(B_alpha)`
`f^-1(nnn_(alpha in I) B_alpha) = nnn_(alpha in I) f^-1(B_alpha)`
`f^-1(Y\\B) = X\\f^-1(B)`

设 `f: X |-> Y`, 记 `"Ker"\ f = {(x_1, x_2) in X xx X: f(x_1) = f(x_2)}` (也就是说, `(x_1, x_2) in "Ker"\ f iff f(x_1) = f(x_2)`) 为 `f` 的核.

设 `f: X |-> Y`, `g: Y |-> W`. 称由 `h(x) = g(f(x))` 定义的 `h: X |-> W` 为 `g` 与 `f` 的复合映射, 记为 `h = g@f`. 映射的复合满足结合律.

设 `f: X |-> Y`, 称 `f` 为一满射, 如果 `AA y in Y`, `EE x in X`, `f(x) = y`; 称 `f` 为一单射, 如果 `f(x_1) = f(x_2)` 蕴含 `x_1 = x_2`. 我们有 `f` 为一满射当且仅当 `"Im"\ f = Y`;
`f` 为一单射当且仅当 `"Ker"\ f = iota_X`, 关于象集, 原象集和单射, 满射的关系, 我们有 (设 A, B 分别为 X, Y 的真子集): `f(f^-1(B)) sube B`, 等号成立当且仅当 `f` 为满射;
`f^-1(f(A)) supe A`, 等号成立当且仅当 `f` 为单射. 另外, `f` 为一单射当且仅当存在 `g: Y |-> X`, `g@f = 1_X`;
`f` 为一满射当且仅当存在 `g: Y |-> X`, `f@g = 1_Y`. 称 `f` 为一双射, 如果 `f` 即是单射, 也是满射. 这时对任意 `y in Y`, 存在唯一 `x in X` 使 `y = f(x)` (其存在性由满射保证, 唯一性由单射保证), 我们有 `f` 为一双射当且仅当存在 `g: Y |-> X`, `g@f = 1_X`, `f@g = 1_B`. 这时记 `g = f^-1` 为 `f` 的逆映射.

`iota_X`: `X xx X` 上的最小等价关系, 每个元素只等价于自身.

特征函数

设 `f: X |-> Y`. 当 `Y = X` 时, 又称 `f` 为 `X` 上的变换. 当 `Y = RR` 时, 一般称 `f` 为函数.

设 `A sube X`, 定义特征函数 `chi_A: X |-> {0, 1}` 如下: `chi_A(x) = { (1, if x in A), (0, if x in X\\A) :}` 一定意义上特征函数可以代表集合 A 本身. 如 `A = B iff chi_A = chi_B`,
`A sube B iff chi_A le chi_B`. 关于特征函数与集合运算之间的联系, 我们有 `chi_(A^C)(x) = 1 - chi_A(x)`
`chi_(A nn B)(x) = chi_A(x) * chi_B(x)`
`chi_(A \\ B)(x) = chi_A(x)(1 - chi_B(x))`
`chi_(A uu B)(x) = chi_A(x) + chi_B(x) - chi_A(x) * chi_B(x)`
`chi_(A /_\ B)(x) = |chi_A(x) - chi_B(x)|`

基数

基数的定义

设 `A, B sube X`. 称集合 A 与 B 对等, 如果存在从 A 到 B 的双射; 这时记 `A~B`. 可以验证, 对等构成 `X` 上的等价关系 (自反性, 对称性, 传递性). 从而我们可以用这一等价关系的等价类来定义基数, 即相互对等的集合具有相同基数, 不相互对等的集合具有不同基数: `A~B iff |A| = |B|`.

(Banach) 若 `f: X |-> Y`, `g: Y |-> X`, 则存在分解 `X = X_1 uu X_2`, `X_1 nn X_2 = O/`;
`Y = Y_1 uu Y_2`, `Y_1 nn Y_2 = O/`. 使得 `f(X_1) = Y_1`, `g(Y_2) = X_2`.

(Cantor-Bernstein) 若集合 `X ~ X_1`, `Y ~ Y_1`, 且 `X_1 sube Y`, `Y_1 sube X`, 则 `X ~ Y`.

若 `A ~ C`, `A sube B sube C`, 则 `A ~ B ~ C`.

设 `A, B sube X`. 由 Cantor-Bernstein 定理知, "A 与 B 的子集对等" 构成 X 上一偏序 (自反性, 反对称性, 传递性): A 对等于 B 的子集 `iff |A| le |B|`.

`f: [-1, 1] |-> (1, -1)` 建立了闭区间与开区间之间的双射, 其中 `f(x) = { (1/(n+1), if x = 1/n;), (-1/(n+1), if x = -1/n;), (x, if else.) :}` 这一做法可以推广到高维的单位球上.

可列集, 可数集

记 `|NN| = aleph_0`. 称 `A` 为可列集, 如果 `A ~ NN`, 亦即 `|A| = aleph_0`. 可列集与有限集合称可数集.

设 `|E| ≮ oo`, 则 `|E| ge aleph_0`.

若 `|A_n| = aleph_0`, `n = 1, 2, cdots`, 则 `|uuu_(n=1)^oo A_n| = aleph_0`.

注意 `B_k = {a_(ij): i + j = k}` 为有限集, 而 `uuu_(n=1)^oo A_n = uuu_(k=2)^oo B_k`.

`NN xx NN = uuu_(m in NN) {(m, n): n in NN}` 是可列集; `|prod_(k=1)^n NN|` 是可列集 (对 n 归纳); `QQ = uuu_(k=1)^oo {n/k: n in ZZ}` 是可列集.

下面是一些可数集 (可列或有限集) 的例子与结论.

由于 `QQ` 在 `RR` 上稠密, 即任意开区间中存在有理数, 我们有如下结论:

  1. `RR` 中互不相交的开区间族是可数集. 事实上, 从每个开区间中可以选出一有理数, 从而这一开区间族对等于 `QQ` 的子集.
  2. `RR` 上单调函数的不连续点集为可数集. 后面将看到, 可数点集是零测集, 故此推论也可写成: `RR` 上的单调函数几乎处处连续.

设 A 为无限集, B 为可数集, 则 `A uu B ~ A`.

`|A| ≮ oo iff EE B sube A, A ~ B`.

连续基数

`RR`, (0, 1], 全体自然数列, 二进制小数... 不可数, 它们与一可列集的幂集对等, 记其基数为 `aleph_1 = 2^(aleph_0)`.

若 `|A_n| = aleph_1`, `n = 1, 2, cdots`, 则 `|uuu_(n=1)^oo A_n| = aleph_1`.

(无最大基数定理) `|X| lt |cc"P"(X)|`. 任一集合都存在幂集, 于是无最大基数.

易知 `X ~ {{x}: x in X} sube cc"P"(X)`, 所以 `|X| le |cc"P"(X)|`. 下证 X 与其幂集不对等. 假设存在双射 (从而是满射) `f: X |-> cc"P"(X)`, 令 `Y = {x in X: x !in f(x)}`, 则 `Y in cc"P"(X)`. 由 `f` 是满射, 存在 `y in X`, 使 `f(y) = Y`. 然而不论 `y in Y` 还是 `y !in Y` 均引出矛盾 (罗素悖论). 故 X 与其幂集不对等.
