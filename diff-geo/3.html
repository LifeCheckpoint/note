曲面的概念

曲面的定义及参数变换

曲面是平面区域 `D` 到 `E^3` 的映射 `bm r(u,v) = (x(u,v), y(u,v), z(u,v))`, 满足

  1. `x, y, z in C^oo(D)`;
  2. `bm r_u = ((del x)/(del u), (del y)/(del u), (del z)/(del u))` 与 `bm r_v = ((del x)/(del v), (del y)/(del v), (del z)/(del v))` 线性无关, 即 `bm r_u ^^ bm r_v != bb 0`.

曲面的参数变换
设 `bm r(u, v): D |-> E^3`, `sigma: bar D |-> D`
`(bar u, bar v) |-> (u, v)`, 其中 `sigma` 一阶连续可微, 且 `|J| = (del(u,v))/(del(bar u, bar v)) != 0`, 则 `bm r` 有新参数表示 `bm r(u(bar u, bar v), v(bar u, bar v))`. 视两个曲面相同, 如果存在满足上述条件的参数变换使它们相同.

设参数变换如前述. 坐标切向量 `bm r_u` 和 `bm r_v` 的变换公式为 `[(bm r_(bar u)), (bm r_(bar v))] = J [(bm r_u), (bm r_v)]`,
`bm r_(bar u) ^^ bm r_(bar v) = |J| bm r_u ^^ bm r_v`.

球面 `x^2 + y^2 + z^2 = a^2` 有球坐标表示 `{( x = a cos u cos v ),( y = a cos u sin v ),( z = a sin u ):}`, `u in [-pi/2, pi/2]`, `v in [0, 2pi]`. 和球极投影坐标表示: `{( x = 2a^2 u // d^2 ),( y = 2a^2 v // d^2 ),( z = a delta^2 // d^2 ):}`, `u, v in RR`. 其中 `d = sqrt(u^2 + v^2 + a^2)` 是 `(u, v)` 到北极的距离, `delta = sqrt(u^2 + v^2 - a^2)` 是 `(u, v)` 到球面的切线长.

旋转面
`xz` 平面上与 `z` 轴无交的曲线 `{(x = x(u)), (z = z(u)):}` 绕 `z` 轴旋转得到 `{(x = x(u) cos v), (y = x(u) sin v), (z = z(u)):}`. 特别当 `x(u) = a cos u`, `z(u) = a sin u` 时, 得到球面.

柱面
`xy` 平面上的曲线 `{(x = x(u)), (y = y(u)):}` 沿 `z` 轴平移得到 `{(x = x(u)), (y = y(u)), (z = v):}`.

直纹面
称直线沿空间曲线 `bm r = bm a(u)` 滑动扫过的曲面 `bm r(u,v) = bm a(u) + v bm b(u)` 为直纹面. 取定 `u` 时, 就确定一条直母线. 当 `bm a` 为常向量时, 为锥面; 当 `bm b/|bm b|` 为常向量时, 为柱面.

切线面
空间曲线 `bm r(u)` 的切线的全体 `bm r(u,v) = bm r(u) + v bm r'(u)`. 切线面也是直纹面的一种.

切向量, 切平面, 法向量

称 `bm r_u`, `bm r_v` 为坐标切向量. 它们线性无关, 从而确定了曲面 `S` 在点 `P_0 = (u, v)` 处的切平面 `T_(P_0)S`. `S` 上过 `P_0` 的任一曲线在 `P_0` 的切向量落在切平面 `T_(P_0)S` 上; 反之, 对切平面 `T_(P_0)S` 上的任一切向量 `bm t`, 存在曲面上过 `P_0` 的一曲线, 它在 `P_0` 的切向量与 `bm t` 同向.

`bm r_u ^^ bm r_v` 是曲面的一个法向量. `bm n = (bm r_u ^^ bm r_v)/|bm r_u ^^ bm r_v|` 是曲面的一个单位法向量.

当曲面由方程 `F(bm r) = c` 给出时, 法向量为 `grad F(bm r_0)`, 切平面为 `(:grad F, bm r - bm r_0:) = 0`.

曲面的切平面, 法线与参数选取无关.

曲面的第一基本形式 (The first fundamental form)

`"I" = "d"s^2 = "d" bm r^2 = (bm r_u "d"u + bm r_v "d" v)^2` `= E "d"u"d"u + 2F "d"u"d"v + G"d"v"d"v = ["d"u, "d"v][(E, F),(F, G)][("d"u),("d"v)]`.

其中 `E = (:bm r_u, bm r_u:)`,
`F = (:bm r_u, bm r_v:)`,
`G = (:bm r_v, bm r_v:)`.

第一基本形式是正定的二次型, 是曲面上的度量. 它与参数选取无关, 在合同变换下不变.

平面 `bm r(u,v) = (u, v, 0)` 的第一基本形式 `"I" = "d"u"d"u + "d"v"d"v`.

柱面 `bm r(u,v) = (x(u), y(u), v)` 的第一基本形式 `"I" = (x'^2 + y'^2)"d"u"d"u + "d"v"d"v`. 取 `u` 是弧长参数时, `"I" = "d"u"d"u + "d"v"d"v`.

球面 `bm r(theta, varphi) = (a cos theta cos varphi, a cos theta sin varphi, a sin varphi)` 的第一基本形式 `"I" = a^2 "d"theta"d"theta + a^2 cos^2 theta "d"varphi"d"varphi`. 球面 `bm r(u,v) = (2a^2u//d^2, 2a^2v//d^2, a(d^2-2a^2)//d^2)` `(d^2 = u^2 + v^2 + a^2)` 的第一基本形式 `"I" = ((2a^2)/d^2)^2 ("d"u"d"u + "d"v"d"v)`.

曲面的第二基本形式 (The second fundamental form)

`"II" = -(:"d"bm r, "d"bm n:) = L"d"u"d"u + 2M"d"u"d"v + N"d"u"d"v = ["d"u, "d"v][(L, M),(M, N)][("d"u),("d"v)]`.

其中 `{:( ,L = (:bm r_(u u)"," bm n:) = -(:bm r_u"," bm n_u:)"," ),( ,M = (:bm r_(uv)"," bm n:) = -(:bm r_u"," bm n_v:) = -(:bm r_v"," bm n_u:)"," ),( ,N = (:bm r_(v v)"," bm n:) = -(:bm r_v"," bm n_v:). ):}`

以上等式可由 `(:bm r_u, bm n:) = (:bm r_v, bm n:) = 0` 求导推得.

第二基本形式在同向参数变换 (`|J| gt 0`) 下不变, 反向则变号; 在同向刚体运动 (`|T| = 1`) 下不变, 反向则变号. 根据曲面在一点处第二基本形式系数矩阵的行列式符号的不同, 将曲面上的点作如下分类:

  1. `LN-M^2 gt 0`, 正定或负定, 凸或凹, 椭圆点;
  2. `LN-M^2 lt 0`, 不定, 马鞍型, 双曲点;
  3. `LN-M^2 = 0`, 退化, 抛物点; 若 `L = M = N = 0`, 为平点.

记 `bm r = bm r(u,v)`, `bm r_0 = bm r(u_0, v_0)`, `bm n_0 = bm n(u_0, v_0)`, 定义曲面高度函数 `f(u, v) = (:bm r - bm r_0, bm n_0:)`, 则 `f_u = (:bm r_u, bm n_0:)`, `f_v = (:bm r_v, bm n_0:)`, `"II" = ["d"u, "d"v] [(f_(u u), f_(uv)), (f_(vu), f_(v v))][("d"u),("d"v)]`.

平面 `bm r(u,v) = (u, v, 0)`. 注意 `bm n` 为常向量, 故 `"II" = -(:"d"bm r, "d"bm n:) = 0`.

柱面 `bm r(u,v) = (x(u), y(u), v)`. 记 `kappa = |(x_u, y_u),(x_(u u), y_(u u))|`, 有 `"II" = -kappa "d"u"d"u`.

球面 `bm r(theta, varphi) = (a cos theta cos varphi, a cos theta sin varphi, a sin theta)`. `"II" = a "d"theta"d"theta + a cos^2 theta "d"varphi"d"varphi`.

法曲率与 Weingarten 变换

法曲率

设曲面 `S` 上有一曲线 `bm r(s)`, `s` 是其弧长参数. `bm r(s)` 在一点的单位切向量为 `("d"bm r)/("d"s)`, 曲率向量为 `("d"^2 bm r)/{:"d"s:}^2`. 由 `(: ("d"bm r)/("d"s), bm n:) = 0` 求导得 ` (: ("d"^2 bm r)/{:"d"s:}^2, bm n :) = -(:("d"bm r)/("d"s), ("d"bm n)/("d"s):) = L(("d"u)/("d"s))^2 + 2M("d"u)/("d"s)("d"v)/("d"s) + N(("d"u)/("d"s))^2`. 记上式的结果为 `k_n`, 它只与曲线在一点处的单位切向量以及曲面的第二基本形式有关, 而与曲线选取无关. 称 `k_n` 为曲面在一点处沿单位切向量 `(("d"u)/("d"s), ("d"v)/("d"s))` 的法曲率.

对任意非零切向量 `bm t = ("d"bm r)/("d"s) = xi bm r_u + eta bm r_v`, 定义 `k_n(bm t) = k_n(bm t/|bm t|) = 1/|bm t|^2 (L xi^2 + 2M xi eta + N eta^2) = (L xi^2 + 2M xi eta + N eta^2)/(E xi^2 + 2F xi eta + G eta^2)`.

从直观上, 法曲率反映了曲面沿给定方向的弯曲程度. 若将曲面沿着该方向 "切开", 则剖面边缘 (视为平面曲线) 的曲率就是 `k_n`. 法曲率在同向参数变换/刚体运动下不变, 反向则变号.

半径为 `a` 的球面: `k_n = 1/a`. 平面: `k_n = 0`. 柱面 `bm r(u,v) = (x(u), y(u), v)`, 其中 `u` 为弧长参数. 这时 `|bm r_u| = |bm r_v| = 1`, 对单位切向量 `bm w = cos theta bm r_u + sin theta bm r_v` (`theta` 是 `bm w` 与 `bm r_u` 夹角), `k_n(bm w) = k_n(theta) = -kappa cos^2 theta`.

Gauss 映射, Weingarten 变换

Gauss 映射定义为曲面 `S` 到 `E^3` 的单位球面 `S^2` 的映射: `bm g: S |-> S^2`
`bm r(u, v) |-> bm n(u, v)`.

Weingarten 变换是 Gauss 映射的导映射 (取反). 它是切平面上的线性变换: `cc W: T_P S |-> T_P S`
`lambda bm r_u + mu bm r_v |-> -(lambda bm n_u + mu bm n_v). 特别, `cc W(bm r_u) = -bm n_u`, `cc W(bm r_v) = -bm n_v`, `cc W("d"bm r) = -"d"bm n`.

也有定义 Weingarten 变换为 `cc W("d"bm r) = "d"bm n` 的.

Weingarten 变换与曲面的参数选取无关.

曲面沿单位切向量 `bm t` 的法曲率 `k_n(bm t) = (:cc W(bm t), bm t:)`.

Weingarten 变换是自共轭变换: `AA bm v, bm w in T_P S`, `(:cc W(bm v), bm w:) = (:cc W(bm w), bm v:)`. 特别取两个向量为切平面的单位正交基底 `bm e_1, bm e_2` 时, 由上式知 Weingarten 变换在正交基底下的矩阵为二阶实对称阵.

Weingarten 变换的矩阵即 Gauss 映射的 Jacobi 阵 (取反).

Weingarten 变换的两个特征值是实数. 它们不相等时, 有两个相互正交的特征向量; 相等时, 任一个切向量都是特征向量.

主曲率

称 Weingarten 变换的两个特征值 `k_1, k_2` 为主曲率, 相应的特征向量为主方向. 称 `K = k_1 k_2` 为 Gauss 曲率, `H = (k_1 + k_2)/2` 为平均曲率.

主曲率实质上是主方向上的法曲率, 是法曲率的特殊情形. 我们有下面的定理:

Euler 公式 设 `k_1, k_2` 为主曲率, `bm e_1, bm e_2` 为相应的主方向, 则法曲率 ` k_n(cos theta bm e_1 + sin theta bm e_2) = k_1 cos^2 theta + k_2 sin^2 theta`. 可以看出, 当 `k_1 != k_2` 时, `k_n` 在主方向上取得极值; `k_1 = k_2` 时, 各方向上的法曲率相等.

计算主曲率

设 Weingarten 变换在基 `(bm r_u, bm r_v)` 下的矩阵为 `bm A = [(a, b),(c, d)]` (未必对称), 即 `cc W(bm r_u) = -bm n_u = a bm r_u + b bm r_v`,
`cc W(bm r_v) = -bm n_v = c bm r_u + d bm r_v`. 以上两式分别与 `bm r_u, bm r_v` 作内积, 得 `[(L, M),(M, N)] = [(a, b), (c, d)][(E, F), (F, G)]`, 即 `[(a, b), (c, d)] = [(L, M),(M, N)][(E, F), (F, G)]^-1`. 主曲率须满足特征方程 `|k bm E - bm A| = 0`, 即 `k^2 - (a+d) k + |bm A| = 0`. 易知 Gauss 曲率 `K = |(a,b),(c,d)| = |(L,M),(M,N)|//|(E,F),(F,G)|`.

`K` 还满足 `bm n_u ^^ bm n_v = (a bm r_u + b bm r_v) ^^ (c bm r_u + d bm r_v) = (ad-bc) bm r_u ^^ bm r_v = K bm r_u ^^ bm r_v`. 直观上, Gauss 映射将曲面上的小块邻域映到单位球面上的区域. 对应区域的面积之比 `K = |bm n_u ^^ bm n_v|/|bm r_u ^^ bm r_v|` 就是 Gauss 曲率.

曲面的第三基本形式定义为 `"III" = (:"d"bm n, "d"bm n:)`, 它也是正定二次型. 可以证明, `K "I" - 2H "II" + "III" = 0`.

曲面举例

旋转曲面

`bm r(u, v) = (f(u) cos v, f(u) sin v, g(u))`, `f gt 0`.

直接计算 `bm r_u = (f'(u)cos v, f'(u)sin v, g'(u))`,
`bm r_v = (-f(u)sin v, f(u)cos v, 0)`,
`E = f'^2+g'^2`, `quad F = 0`, `quad G = f^2`,
`L = (f' g'' - f'' g')/sqrt E`, `quad M = 0`, `quad N = (fg')/sqrt E`,
`k_1 = L/E`, `quad k_2 = N/G`, `quad K = k_1 k_2`, `quad H = (k_1 + k_2)/2`. 若取 `u` 为 `xz` 平面上曲线 `(f(u), g(u))` 的弧长参数, 则 `f'^2 + g'^2 = 1`, `f'f'' + g'g'' = 0`, 因此 ` f'g'' - f''g' = (f'g'' - f''g') (g')/(g') = (f'(g''g') - f''g'^2)/(g')` `= (-f'f'f'' - f''(1-f'^2))/(g') = -(f'')/(g')`. 上面各式有比较简单的表达式 `E = 1`, `quad F = 0`, `quad G = f^2`,
`L = -(f'')/(g')`, `quad M = 0`, `quad N = fg'`,
`k_1 = -(f'')/(g')`, `quad k_2 = (g')/f`, `quad K = -(f'')/f`, `quad H = 1/2 ((g')/f - (f'')/(g'))`.

下面我们总是取 `u` 为 `(f(u), g(u))` 的弧长参数, 以简化计算.

若曲面在一点的两个主曲率相等, 则它在这一点第二基本形式与第一基本形式的系数成比例, 即 `L/E = M/F = N/G = k`, 则称该点为曲面的脐点. 特别 `k != 0` 时, 称为圆点. `k = 0` 时称为平点.
