三角函数与反三角函数

勾股定理

`sin^2 x + cos^2 x = 1`;
`tan^2 x + 1 = 1 // cos^2 x`;
`1 + cot^2 x = 1 // sin^2 x`.

和角公式

`sin (x +- y) = sin x cos y +- cos x sin y`;
`cos (x +- y) = cos x cos y ∓ sin x sin y`;
`tan (x +- y) = (tan x +- tan y) / (1 ∓ tan x tan y)`.

倍角/半角公式

运用和角公式推出

`cos 2x = cos^2 x - sin^2 x` `= 1 - 2 sin^2 x` `= 2 cos^2 x -1`;
`sin 2x = 2 sin x cos x`;
`cos 3x = 4 cos^3 x - 3 cos x`;
`sin 3x = 3 sin x - 4 sin^3 x`;
`cos^2 x = (1 + cos 2x)//2`;
`sin^2 x = (1 - cos 2x)//2`;
`tan^2 x = (1 - cos 2x)/(1 + cos 2x)`;
`tan x = (sin 2x)/(1 + cos 2x) = (1 - cos 2x)/(sin 2x)`;
`1 +- sin 2x = (sin x +- cos x)^2`.

和差化积

运用和角公式推出

`sin x +- sin y = 2 sin{:(x+-y)/2:} cos{:(x∓y)/2:}`;
`cos x + cos y = 2 cos{:(x+y)/2:} cos{:(x-y)/2:}`;
`cos x - cos y = 2 sin{:(x+y)/2:} sin{:(x-y)/2:}`.

积化和差

运用和角公式推出

`2 sin x cos y = sin(x+y) + sin(x-y)` `= sin(y+x) - sin(y-x)`;
`2 cos x cos y = cos(x-y) + cos(x+y)`;
`2 sin x sin y = cos(x-y) - cos(x+y)`.

辅助角公式

`A sin x + B cos x = sqrt(A^2+B^2) sin(x+varphi)`, `quad tan varphi = B/A`;
`(1 + C tan x)/(C - tan x) = tan (x+varphi)`, `quad cot varphi = C`.

万能代换

令 `t = tan {:x/2:}`, 有 `x = 2 arctan t`, 从而 `dx = (2 dt)/(1+t^2)`, `quad sin x = (2t)/(1+t^2)`, `quad tan x = (2t)/(1-t^2)`, `quad cos x = (1-t^2)/(1+t^2)`.

反三角函数

定义域, 值域: `y = arcsin x`, `quad x in [-1, 1]`, `y in [-pi/2, pi/2]`;
`y = arccos x`, `quad x in [-1, 1]`, `y in [0, pi]`;
`y = arctan x`, `quad x in RR`, `y in (-pi/2, pi/2)`;
`y = "arccot" x`, `quad x in RR`, `y in (0, pi)`.

因为 `arcsin x`, `arccos x`, `arctan x` 都在各自的定义域上单调, 所以对所有定义域中的 `x`, `sin arcsin x = x`, `quad cos arccos x = x`, `quad tan arctan x = x`. 反之, 则不那么优雅: `arcsin sin x = (-1)^n (x-n pi)`, `quad |x-n pi| le pi/2;
`arctan tan x = x-n pi`, `quad |x - n pi| lt pi/2`;
`arccos cos x = { (x-n pi, if n " is even"), ((n+1)pi-x, if n " is odd") :}`, `quad |x-(n+1/2)pi| le pi/2`.

相应于一些三角函数公式, 对于反三角函数有如下恒等式 `arcsin x + arccos x = pi/2`;
`arctan x + text(arccot) x = pi/2`;
`2 arctan "e"^x - arctan sinh x = pi/2`;
 以及 ` arcsin x = text(sgn) x * arccos sqrt (1 - x^2) = arctan {: x/sqrt (1 - x^2) :}`;
`arctan x = sgn x * arccos{:1/sqrt(1 + x^2):}` `= arcsin {:x/sqrt(1 + x^2):}`;
`arccos x = { (arcsin sqrt(1-x^2), if x ge 0), (pi - arcsin sqrt(1-x^2), if x lt 0) :} = { (arctan{:sqrt(1-x^2)/x:}, if x ge 0), (pi + arctan{:sqrt(1-x^2)/x:}, if x lt 0) :}`. 还有 ` 2 arctan sqrt ((a-b)/(a+b)) = arccos {: b/a :}`, `quad |a| ge |b|`;
` arccos x = 2 arccos sqrt((1+x)/2) = 2 arcsin sqrt((1-x)/2)`.

下面三个式子常用于数学分析: `arcsin x - arcsin y = { (arcsin(x sqrt(1-y^2) - y sqrt(1-x^2)) if xy ge 0 or x^2 + y^2 le 1), (pi*sgn(x-y) - arcsin(x sqrt(1-y^2) - y sqrt(1-x^2)) if "else") :}`
`arctan x - arctan y = { (arctan{:(x-y)/(1+xy):}, if xy ge -1), (arctan{:(x-y)/(1+xy):} + pi*sgn(x-y), if xy lt -1) :}`;
`arccos x - arccos y = arccos(xy + sqrt((1-x^2)(1-y^2)) ) * sgn(y-x)`.

考虑如下的证明思路 (以 `arcsin x + arccos x = pi/2` 的证明为例): 原式成立当且仅当 `arcsin x = pi/2 - arccos x`, 注意到上式两边之差的绝对值不超过 `2 pi`, 所以上式成立当且仅当 `sin arcsin x = sin (pi/2 - arccos x)`, 即 `x = cos arccos x`, 这显然成立.

双曲三角函数与反双曲三角函数

和角

`sinh(x+-y) = sinh x cosh y +- sinh y cosh x`;
`cosh(x+-y) = cosh x cosh y +- sinh x sinh y`;
`tanh(x+-y) = (tanh x +- tanh y)/(1+-tanh x tanh y)`.

倍角公式

`sinh 2x = 2 sinh x cosh x`;
`cosh 2x = cosh^2 x + sinh^2 x` `= 2 cosh^2 x -1` `= 2 sinh^2 x +1`.

勾股定理

`cosh^2 x = sinh^2 x + 1`;
`1 = tanh^2 x + 1//cosh^2 x`;
`1//tanh^2 x = 1 + 1//sinh^2 x`.

反双曲三角函数

`"arsh "x = ln (x + sqrt(x^2+1))`, `quad x in RR`;
`"arch "x = ln (x + sqrt(x^2-1))`, `quad x ge 1`;
`"arth "x = 1/2 ln ((1+x)/(1-x))`, `quad -1 lt x lt 1`.

De Moivre 定理

`AA n in ZZ`, 对 `n` 作归纳即可证明, `(cos x + "i"sin x)^n = cos nx + "i" sin nx`. 上式左边展开, 分别取实部和虚部就得到: `AA n in ZZ^+`. `cos n x = sum_(j=0)^k (-1)^j ((n),(2j)) cos^(n-2j)x sin^(2j)x`, `quad n = 2k` 或 `2k+1`;
`sin n x = sum_(j=1)^k (-1)^(j-1) ((n),(2j-1)) cos^(n-2j+1)x sin^(2j-1)x`. `quad n = 2k` 或 `2k-1`.

Euler 公式

`"e"^(ix) = cos x + "i" sin x`, `quad AA x in CC`. 从而 `cos x = ("e"^(ix) + "e"^(-ix))/2`, `quad sin x = ("e"^(ix) - "e"^(-ix))/{2i}`. 且 `arccos x = -"i" ln (x + "i"sqrt(1-x^2))`,
`arcsin x = -"i" ln (sqrt(1-x^2) + "i"x)`,
`arctan x = -"i" ln {:(1 + "i"x)/sqrt(1+x^2):}`.

比如, 设 `y = arccos x`, 则 `"e"^("i"y) = cos y + "i"sin y = x + "i"sqrt(1-x^2)`, 于是 `y = -"i"ln (x + "i"sqrt(1-x^2))`.

杂例

证明 `x != 2m pi`, `m in ZZ` 时, `sum_(k=1)^n cos kx` `= (cos {:((n+1)x)/2:} sin {:(nx)/2:}) / sin{:x/2:}`;
` sum_(k=1)^n sin kx = (sin {:((n+1)x)/2:} sin {:(nx)/2:}) / sin {:x/2:}`. 从而对 `AA n in ZZ^+`, `sum_(k=1)^n cos {:(2k pi)/n:}` `= sum_(k=1)^n sin {:(2k pi)/n:} = 0`. 这个等式的几何意义是, 向量组 `bm e_k = ( cos {:(2k pi)/n:}, sin {:(2k pi)/n:} )`, `k = 1, 2, cdots, n`, 中所有向量之和为零.

只证第一式. 第二式类似. 利用积化和差与和差化积: `2 sin{:x/2:} sum_(k=1)^n cos kx` `= sum_(k=1)^n [sin(k+1/2)x - sin(k-1/2)x]` `= sin(n+1/2)x - sin{:x/2:}` `= 2 cos {:((n+1)x)/2:} sin {:(nx)/2:}`.
