函数的极限

邻域与函数极限

    邻域
  1. 设 `x_0 in RR`, `delta gt 0`. 称 `(x_0-delta, x_0)` 为 `x_0` 的左 `delta` 邻域, `(x_0, x_0+delta)` 为 `x_0` 的右 `delta` 邻域, `B(x_0, delta) := (x_0-delta, x_0+delta)` 称为 `x_0` 的 `delta` 邻域, `overset @ B(x_0, delta)` `:= B(x_0, delta)\\{x_0}` `= (x_0-delta, x_0) uu (x_0, x_0+delta)` 为 `x_0` 的去心 `delta` 邻域. 显然 `x in overset @ B(x_0, delta) iff 0 lt |x-x_0| lt delta`.
  2. 设 `delta gt 0`, 称 `(1//delta, +oo)` 为 `oo` 的左 `delta` 邻域, `(-oo, -1//delta)` 为 `oo` 的右 `delta` 邻域. `overset @ B(oo, delta)` `:= (-oo, -1//delta) uu (1//delta, +oo)` 称为 `oo` 的去心 `delta` 邻域.

函数极限 设函数 `f` 在点 `x_0 in RR uu {oo}` 的某个去心邻域内有定义, `a in RR`, 如果对任意 `epsi gt 0`, 都存在相应的 `delta gt 0`, 使得 `x in overset @ B(x_0, delta)` 时, 成立 `|f(x)-a| lt epsi`, 则称 `f(x)` 在 `x` 趋于 `x_0` 时以 `a` 为极限, 或称当 `x` 趋于 `x_0` 时, `f(x)` 趋于 `a` 或收敛于 `a`, 记作 `lim_(x to x_0) f(x) = a` 或 `f(x) to a` (`x to x_0`). 函数极限的这种定义称为 `epsi`-`delta` 语言.

有界函数, 无穷大与无穷小

设 `f(x)` 在点集 `E sube RR` 上有定义, 且存在 `M gt 0` 使得 `|f(x)| le M`, `quad AA x in E`, 则称 `f(x)` 在 `E` 上有界, 否则称它在 `E` 上无界.
如果存在 `x_0` 的某去心邻域 `overset @ B(x_0, delta) sube E`, 使得 `|f(x)| le M`, `quad AA x in overset @ B(x_0, delta)`, 则称 `f(x)` 在 `E` 上局部有界.
在整个定义域上有界的函数称为有界函数, 否则称为无界函数.

函数极限的性质

函数极限的唯一性 若极限 `lim_(x to x_0) f(x)` 存在, 则该极限唯一.

反设 `x to x_0` 时, `f(x)` 有极限 `a` 和 `b`, 且 `a != b`, 不妨设 `a lt b`. 令 `epsi = (b-a)//2 gt 0`, 由极限定义, 存在 `delta_1, delta_2 gt 0`, 使得 `a-epsi lt f(x) lt a+epsi`, `quad AA x in overset @ B(x_0, delta_1)`,
`b-epsi lt f(x) lt b+epsi`, `quad AA x in overset @ B(x_0, delta_2)`. 取 `delta = min{delta_1, delta_2}`, 从而当 `x in overset @ B(x_0, delta)` `= overset @ B(x_0, delta_1) nn overset @ B(x_0, delta_2)` 时, `f(x) lt a + epsi = b - epsi lt f(x)`, 矛盾. 所以 `a = b`.

局部有界性 若极限 `lim_(x to x_0) f(x)` 存在, 则 `f(x)` 在 `x_0` 附近局部有界.

设 `lim_(x to x_0) f(x) = a`. 由极限定义, 存在 `delta gt 0`, 使得 `a - 1 lt f(x) lt a + 1`, `AA x in overset @ B(x_0, delta)`. 取 `M = |a| + 1`, 则 `|a+-1| le M`, 于是 `|f(x)| lt M`, `quad AA x in overset @ B(x_0, delta)`.

    局部保号性 设 `lim_(x to x_0) f(x) = a != 0`, 则存在一个介于 `a` 和 `0` 之间的实数 `r`,
  1. `a gt 0` 时, 在 `x_0` 的某去心邻域内有 `f(x) gt r gt 0`;
  2. `a lt 0` 时, 在 `x_0` 的某去心邻域内有 `f(x) lt r lt 0`.
    3. 从而若函数趋于 `x_0` 时有一个不等于零的极限, 则在 `x_0` 的某去心邻域内, `f(x)` 与 `0` 保持一个正的距离.

只证结论 1, 结论 2 证明类似. 取 `epsi = a/2 gt 0`, 由极限定义, 在 `x_0` 的某去心邻域内有 `a-a/2 lt f(x) lt a+a/2`, 从而 `f(x) gt a/2 gt 0`. 取 `r = a/2` 即可.

这些定理的证明完全类似于数列极限的对应定理. 下面不加证明, 仅罗列结论.

函数极限的保序性 如果 `f(x)`, `g(x)` 在 `x to x_0` 时有极限, 且在 `x_0` 的某去心邻域内成立 `f(x) le g(x)`, 则 `lim_(x to x_0) f(x) le lim_(x to x_0) g(x)`.

两边夹法则 设函数 `f`, `g`, `h` 都在 `x_0` 的某去心邻域内有定义, 且在该邻域内成立 `f(x) le g(x) le h(x)`. 又设 `lim_(x to x_0) f(x) = lim_(x to x_0) h(x) = a`, 则 `lim_(x to x_0) g(x) = a`.

函数极限的四则运算 若 `lim_(x to x_0) f(x) = a`, `lim_(x to x_0) g(x) = b`, 则 `lim_(x to x_0) (f(x) +- g(x)) = a +- b`, `quad lim_(x to x_0) (f(x) g(x)) = a b`, `quad lim_(x to x_0) (f(x))/(g(x)) = a/b` (当 `b != 0`). 从而对任意常数 `c` 有 `lim_(x to x_0) (c f(x)) = c a`.

复合函数的极限

求 `root 3 (1+2x^2+x^3)` 在 `x to oo` 时的渐近线.

设它有渐近线 `y = a x + b`, 则 `0 = lim_(x to oo) (root 3 (1+2x^2+x^3) - a x - b)` `= lim_(x to 0) (root 3 (1 + 2/x^2 + 1/x^3) - a/x - b)` `= lim_(x to 0) (root 3 (x^3 + 2x + 1) - a - b x)//x` `= lim_(x to 0) (1 + 1/3(x^3 + 2 x) + o(x) - a - b x)//x` `= lim_(x to 0) (2/3-b) + (1-a)//x`. 因此 `a = 1, b = 2/3`.

函数的连续性

  1. `a^x` 是 `RR` 上的连续函数, `a gt 0`, `a != 1`;
  2. `log_a x` 是 `(0, +oo)` 上的连续函数, `a gt 0`, `a != 1`;
  3. `sin x`, `cos x` 是 `RR` 上的连续函数;
  4. `arcsin x` 是 `[-1, 1]` 上的连续函数;
  5. `arctan x` 是 `RR` 上的连续函数.
  1. 只证 `a gt 1` 的情形. `AA x_0 in RR`, `AA epsi gt 0`, 由于 `|a^x - a^(x_0)| lt epsi` `iff a^(x_0)-epsi lt a^x lt a^(x_0)+epsi` `iff 1-a^(-x_0)epsi lt a^(x-x_0) lt 1+a^(-x_0)epsi` `iff log_a(1-a^(-x_0)epsi) lt x-x_0 lt log_a(1+a^(-x_0)epsi)`, 令 `delta = min{|log_a(1-a^(-x_0)epsi)|, |log_a(1+a^(-x_0)epsi)|} gt 0`, 则 `0 lt |x-x_0| lt delta` 时, 就有 `|a^x - a^(x_0)| lt epsi`.
  2. 只证 `a gt 1` 的情形. `AA x_0 gt 0`, `AA epsi gt 0`, 由于 `|log_a x - log_a x_0| lt epsi` `iff log_a x_0 - epsi lt log_a x lt log_a x_0 + epsi` `iff x_0 a^-epsi lt x lt x_0 a^epsi` `iff x_0(a^-epsi-1) lt x-x_0 lt x_0(a^epsi-1)`, 令 `delta = min{|x_0 (a^-epsi-1)|, |x_0(a^epsi-1)|} gt 0`, 如果进一步要求 `delta lt x_0`, 则 `0 lt |x-x_0| lt delta` 时, `x gt 0`, 故 `loa_a x` 有定义, 且 `|log_a x - log_a x_0| lt epsi`.
  3. `AA x_0 in RR`, 记 `a = (x+x_0)/2`, `b = (x-x_0)/2`, 运用和差化积公式, `|sin x - sin x_0|` `= |sin(a+b) - sin(a-b)|` `= 2 |cos a sin b|` `le 2 |sin b| le 2 |b|` `= |x-x_0|`. `AA epsi gt 0`, 取 `delta = epsi`, 则 `0 lt |x-x_0| lt delta` 时, `|sin x - sin x_0| lt epsi`.
  4. `AA x_0 in (-1, 1)`, 先证 `lim_(x to x_0) sqrt(1-x^2) = sqrt(1-x_0^2)`. 由于 `|sqrt(1-x^2)-sqrt(1-x_0^2)|` `= |x^2-x_0^2|/(sqrt(1-x^2)+sqrt(1-x_0^2))` `le |x^2-x_0^2|/sqrt(1-x_0^2)` `to 0`, 从而 `lim_(x to x_0) sqrt(1-x^2) = sqrt(1-x_0^2)`. 计算, 由极限的四则运算知 `|arcsin x - arcsin x_0|` `le |tan(arcsin x - arcsin x_0)|` `= |(sin(arcsin x - arcsin x_0))/(cos(arcsin x - arcsin x_0))|` `= |(x sqrt(1-x_0^2) - x_0 sqrt(1-x^2))/(sqrt(1-x^2)sqrt(1-x_0^2) + x x_0)|` `to 0`. 再考虑端点处的左, 右极限...???
  5. 应用极限的四则运算, `|arctan x - arctan x_0|` `le |tan(arctan x - arctan x_0)|` `= |x-x_0|/|1+x x_0|` `to 0`.
