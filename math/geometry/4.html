Apollonius 圆 (阿氏圆) 平面上到两定点距离之比为常数 `k` (`k gt 0`, `k != 1`) 的点的轨迹为圆.

设这两个定点为 `A, B`, 线段 `AB` 长为 `l`. `P`, `Q` 分别是线段 `AB` 的内, 外分点, 满足 `AP = k/(k+1) l`, `quad BP = 1/(k+1) l`,
`AQ = k/(k-1) l`, `quad BQ = 1/(k-1) l`. 假设平面上一点 `R` 满足 `(AR)/(BR) = k`, 则作相似三角形 `triangle ORA S~ triangle OBR`, 相似比为 `k`. 记 `r = OR`, 由相似关系得 `(OA)/(OR) = k`, `quad (OR)/(OB) = k`. 于是 `l + OB = OA = k^2 OB`, 解得 `OB = l/(k^2-1)`, `quad r = k OB = k/(k^2-1) l`. 计算知 `OP = OQ = r`, 因此 `R` 落在以 `P, Q` 为直径的圆上.
