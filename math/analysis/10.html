无穷级数的基本概念

无穷级数, 简称级数, 是用加号把数列 `{u_n}` 中的所有数依次连接起来所得到的数学表达式: `sum_(n=1)^oo u_n = u_1 + u_2 + cdots + u_n + cdots`. 每个加数称为级数的一项, 一般项 `u_n` 称为级数的通项. 对每个正整数 `n`, 由级数的前 `n` 项和 `S_n = sum_(k=1)^n u_k` 组成一个数列 `{S_n}_(n=1)^oo`, 称为级数的部分和 (数列). 当部分和数列收敛时, 则称这个级数收敛, 并称部分和数列的极限为级数的和. 故对收敛级数, 我们有 `sum_(k=1)^oo u_n = lim_(n to oo) sum_(k=1)^n u_k`. 另一方面, 当部分和数列发散时, 就称级数发散. 如果级数的部分和数列趋于正 (负) 无穷大, 我们也称这个级数发散到正 (负) 无穷大, 记为 `sum_(n=1)^oo u_n = +oo (-oo)`.

小数就是级数的一个例子. 比如 `3.1415926... = 3 + 1/10 + 4/10^2 + 1/10^3 + 5/10^4 + cdots`. 实际运用中, 许多问题无法求出或不需要求出精确解, 我们就常用级数估计它, 这和用小数近似估计一个实数是相同的思想.

几何级数 `sum_(n=0)^oo x^n` `{ = (1-x)^-1, if |x| lt 1; = oo, if |x| gt 1; 发散, if x = -1; :}`.

  1. `sum_(n=1)^oo c u_n = c sum_(n=1)^oo u_n`;
  2. `sum_(n=1)^oo (u_n +- v_n) = sum_(n=1)^oo u_n +- sum_(n=1)^oo v_n`.
    3. 这两个等式表示右端级数收敛时, 左端级数也收敛, 且它们的和相等. 这个定理表明, 收敛级数可以进行线性运算.

考察两个级数的部分和, 有 `sum_(k=1)^n c u_k = c sum_(k=1)^n u_k`. 在等式两边令 `n to oo`, 就证明了结论 1. 结论 2 的证明类似. 级数的线性可加性是由数列极限的线性可加性保证的.

    应用反证法, 立即得到下面的推论:
  1. 若 `sum_(n=1)^oo u_n` 发散, 则对任意 `c != 0`, `sum_(n=1)^oo c u_n` 发散.
  2. 若 `sum_(n=1)^oo u_n` 发散, `sum_(n=1)^oo v_n` 收敛, 则 `sum_(n=1)^oo (u_n +- v_n)` 发散;

任意改变级数中有限项的数值, 不改变级数的敛散性; 换言之, 对两个级数 `sum_(n=1)^oo u_n`, `sum_(n=1)^oo v_n`, 如果存在正整数 `N`, 使对所有 `n gt N` 的项都有 `u_n = v_n`, 则它们的敛散性相同.

比较它们的部分和, 有 `sum_(k=1)^n u_n` `= sum_(k=1)^N (u_k - v_k) + sum_(k=1)^n v_n`. 由数列极限的运算法则知道, 这两个级数敛散性相同.

由数列收敛的 Cauchy 准则立即推出:

级数收敛的 Cauchy 准则 级数 `sum_(n=1)^oo u_n` 收敛的充要条件是 `AA epsi gt 0`, `EE N in NN`, `|sum_(k=1)^p u_(n+k)| lt epsi`, `quad AA n gt N`, `AA p in NN`.

称级数 `sum_(n=1)^oo u_n` 绝对收敛, 如果 `sum_(n=1)^oo |u_n|` 收敛; 称级数 `sum_(n=1)^oo u_n` 条件收敛, 如果它自身收敛, 但是不绝对收敛.

级数绝对收敛蕴含它收敛.

应用 Cauchy 准则和三角不等式 `|sum_(k=1)^p u_(n+k)| le sum_(k=1)^p |u_(n+k)|` 即可.

级数收敛的必要条件是其通项趋于零.

这当然是 Cauchy 收敛准则的推论, 但也可以这样证: 记 `S_n` 是级数 `sum_(n=1)^oo u_n` 的前 `n` 项和, 则 `u_n = S_n - S_(n-1)`, `n = 2, 3, cdots`. `lim_(n to oo) u_n = lim_(n to oo) S_n - lim_(n to oo) S_(n-1) = 0`.

设级数 `sum_(n=1)^oo u_n`, `sum_(n=1)^oo v_n` 都收敛, 且通项 `u_n ge v_n`, `n = 1, 2, cdots`. 显然 `sum_(n=1)^oo (u_n - v_n)` 收敛, 记其和为 `S`. 如果存在 `k in ZZ^+` 使 `u_k gt v_k`, 那么断言 `S gt 0`.

`S = sum_(n=1)^oo (u_n-v_n)` `= sum_(n=1)^k (u_n-v_n) + sum_(n=k+1)^oo (u_n-v_n)` 第一项大于 0, 第二项由极限的保序性非负, 因此 `S gt 0`.

定积分中也有类似于上例的结论, 但其证明并不简单.

正项级数

正项级数的概念

称一个级数为正项级数, 如果它的每一项的都非负. 注意, 并不要求每一项都是正数.

从上节知道, 一个级数绝对收敛可以保证它自身收敛, 而级数 `sum_(n=1)^oo |u_n|` 的通项是非负的, 因此正项级数的敛散性理论可以用来判断一个级数是否绝对收敛. 另外, 由于去掉级数的有限项不改变其敛散性, 因此对除有限项外各项都是非负的级数, 也可以使用正项级数的敛散性理论.

由数列收敛的单调有界原理立即得到:

正项级数收敛的充要条件是其部分和数列有上界.

下面介绍的所有正项级数的判别法, 都从这个定理出发. 在介绍其它判别法之前, 我们先应用这个定理讨论 `p` 级数的敛散性.

正项级数的敛散性判别

    比较判别法一 设有正项级数 `sum_(n=1)^oo u_n` 和 `sum_(n=1)^oo v_n`, 且存在正整数 `N` 使 `u_n le v_n`, `AA n ge N` (注意到常数因子不改变级数敛散性, 这个条件就是说 `u_n = O(v_n)`), 则
  1. `sum_(n=1)^oo v_n` 收敛 `rArr sum_(n=1)^oo u_n` 收敛;
  2. `sum_(n=1)^oo u_n` 发散 `rArr sum_(n=1)^oo v_n` 发散;

因为去掉级数的有限项不改变级数的敛散性, 所以不妨设 `N = 1`. 由条件知 `sum_(k=1)^n u_n le sum_(k=1)^n v_n`, `quad n = 1, 2, cdots`. 如果 `sum_(n=1)^oo v_n` 收敛, 则 `sum_(n=1)^oo u_n` 的部分和有上界, 从而收敛; 反过来, 如果 `sum_(n=1)^oo u_n` 发散, 则其部分和无上界, 于是 `sum_(n=1)^oo v_n` 的部分和也无上界, 是发散的.

    比较判别法二 设有正项级数 `sum_(n=1)^oo u_n` 和 `sum_(n=1)^oo v_n`, 其中 `v_n` 的各项严格大于零, 且存在 `0 le l le +oo` 使成立 `lim_(n to oo) u_n/v_n = l`. 则
  1. 如果 `sum_(n=1)^oo v_n` 收敛, 且 `l lt +oo`, 则 `sum_(n=1)^oo u_n` 也收敛;
  2. 如果 `sum_(n=1)^oo v_n` 发散, 且 `l gt 0`, 则 `sum_(n=1)^oo u_n` 也发散.

`l lt +oo` 时, 有 `u_n = O(v_n)`, `l gt 0` 时, 有 `v_n = O(u_n)`; 再应用比较判别法一即得结论.

    比较判别法三 设有正项级数 `sum_(n=1)^oo u_n` 和 `sum_(n=1)^oo v_n`, 两个级数的各项都严格大于零, 且存在正整数 `N` 使 `u_(n+1)/u_n le v_(n+1)/v_n`, `AA n ge N`. 则
  1. `sum_(n=1)^oo v_n` 收敛 `rArr sum_(n=1)^oo u_n` 收敛;
  2. `sum_(n=1)^oo u_n` 发散 `rArr sum_(n=1)^oo v_n` 发散;

由条件知对任意 `n gt N` 的项都成立 `u_n/u_N = prod_(k=N+1)^n u_k/u_(k-1)` `le prod_(k=N+1)^n v_k/v_(k-1) = v_n/v_N`. 即 `u_n le u_N/v_N v_n`, `AA n gt N`. 应用比较判别法一即得结论.

    D'Alembert 检比法 设 `sum_(n=1)^oo u_n` 是各项严格大于零的正项级数, 且存在 `0 le l le +oo` 使成立 `lim_(n to oo) u_(n+1)/u_n = l`. 则
  1. `l lt 1` 时级数收敛;
  2. `l gt 1` 时级数发散.
    3. 但是 `l = 1` 时, 级数的敛散性无法用此定理判断.
    我们通过与几何级数进行比较, 来证明这个定理.
  1. 先设 `0 le l lt 1`. 取实数 `r` 使得 `l lt r lt 1`, 再令 `v_n = r^n`, `n = 1, 2, cdots`. 则 `sum_(n=1)^oo v_n` 收敛. 注意到 `v_(n+1)/v_n = r gt l = lim_(n to oo) u_(n+1)/u_n`, 所以存在正整数 `N` 使 `u_(n+1)/u_n le v_(n+1)/v_n`, `AA n ge N`. 应用比较判别法三即知 `sum_(n=1)^oo u_n` 收敛.
  2. 再设 `1 lt l le +oo`. 取实数 `r` 使得 `1 lt r lt l` (`l = +oo` 时, 取 `r gt 1` 即可), `类似可证 sum_(n=1)^oo u_n` 发散.
    Cauchy 检根法 设 `sum_(n=1)^oo u_n` 是正项级数, 且存在 `0 le l le +oo` 使上极限 `underset(n to oo) bar lim root n (u_n) = l`. 则
  1. `l lt 1` 时级数收敛;
  2. `l gt 1` 时级数发散.
    3. 但是再一次, `l = 1` 时, 级数的敛散性无法用此定理判断.
  1. 先设 `0 le l lt 1`. 取实数 `r` 使得 `l lt r lt 1`, 则由条件知存在正整数 `N`, 使当 `n ge N` 时 `root n (u_n) le r`. 即 `u_n le r^n`. 由 `r lt 1` 知 `sum_(n=1)^oo r^n` 收敛, 从而 `sum_(n=1)^oo u_n` 也收敛.
  2. 再设 `1 lt l le +oo`. 取实数 `r` 使得 `1 lt r lt l`. 由条件知, 存在 `{u_n}_(n=1)^oo` 的子列 `{u_(n_k)}_(k=1)^oo` 使得对任意正整数 `k` 成立 `root (n_k) (u_(n_k)) ge r`, 即 `u_(n_k) ge r^(n_k)`. 因为 `r gt 1`, 所以 `k to oo` 时, `r^(n_k) to +oo`, 于是 `u_(n_k) to +oo`. 级数 `sum_(n=1)^oo u_n` 的通项不趋于零, 所以它发散.

`l = 1` 时, D'Alembert 和 Cauchy 判别法均失败. 为了判断更精细的级数的敛散性, 可以考虑下一节的 Gauss 判别法.

`p` 级数与调和级数

`p` 级数 `sum_(n=1)^oo 1/n^p` 在 `p gt 1` 时收敛, `p le 1` 时发散.

    `p le 0` 时, 级数通项不趋于零, 因而发散. 下设 `p gt 0`. 我们通过适当放缩, 将 `p` 级数与几何级数作比较.
  1. 若 `0 lt p le 1`, 对任意正整数 `n ge 2`, 存在正整数 `k` 使得 `2^k le n lt 2^(k+1)`. 现在分析 `p` 级数的部分和, 将它从第二项起, 依次按 `1, 2, 4, 8, cdots, 2^(k-1)` 项组合起来, 并舍弃从 `2^k + 1` 项开始的后面各项: `sum_(i=1)^n i^-p` `ge 1 + sum_(j=1)^k sum_(i=2^(j-1)+1)^(2^j) i^-p` `ge 1 + sum_(j=1)^k 2^(j-1) * 2^(-pj)` `= 1 + 1/2 sum_(j=1)^k 2^((1-p)j)` `= { 1 + a/2 (1-a^k)/(1-a), if 0 lt p lt 1; 1 + k/2, if p = 1; :}`. 其中 `a = 2^(1-p) gt 1`. 注意 `n to oo` 时, `k to oo`, 由上述不等式即知 `p` 级数发散.
  2. `p gt 1` 时, 类似有 `sum_(i=1)^n i^-p` `le sum_(j=0)^k sum_(i=2^j)^(2^(j+1)-1) i^-p` `le sum_(j=0)^k 2^j * 2^(-pj)` `= sum_(j=0)^k 2^((1-p)j)` `= (1-a^(k+1))/(1-a)` `le 1/(1-a)`. 其中 `0 lt a = 2^(1-p) lt 1`. 从而部分和有界, 所以 `p` 级数收敛.
  1. 先设 `p = 1`, 此时由熟知的不等式 (见第二章) `1/(n+1) lt ln(1+1/n) lt 1/n` 立即得到 `sum_(k=1)^n 1/k gt sum_(k=1)^n [ln(k+1) - ln k]` `= ln(n+1)`, 因而级数发散.
  2. `p != 1` 时, 对函数 `x^(1-p)` (它和 `x^-p` 的原函数只相差一个常数因子) 在区间 `[n, n+1]` 上应用微分中值定理得 `(n+1)^(1-p) - n^(1-p) = (1-p) xi^-p`, 其中 `n lt xi lt n+1`. 从而 `1/(n+1)^p lt ((n+1)^(1-p) - n^(1-p))/(1-p) lt 1/n^p`. 因此 `p lt 1` 时, `sum_(k=1)^n 1/k^p` `gt sum_(k=1)^n ((k+1)^(1-p) - k^(1-p))/(1-p)` `= ((n+1)^(1-p) - 1)/(1-p)`, 推知级数发散, 而 `p gt 1` 时, `sum_(k=1)^n 1/k^p` `= 1 + sum_(k=1)^(n-1) 1/(k+1)^p` `lt 1 + sum_(k=1)^(n-1) ((k+1)^(1-p) - k^(1-p))/(1-p)` `= 1 + (n^(1-p) - 1)/(1-p)` `lt 1 - 1/(1-p)`, 表明部分和有界, 所以级数收敛.

`p = 1` 时, `p` 级数又称为调和级数. 我们现在知道它是发散的. 下面的定理估计了它的发散速度.

利用 `p` 级数敛散性的第二种证明的思路, 可以得到更精细的级数的敛散性. 如 `sum_(n=2)^oo 1/(n ln^p n)` 在 `p gt 1` 时收敛, `p le 1` 时发散. 一般地, 记 `ln_0 x = x`, `ln_m x = ln ln_(m-1) x`, `m = 1, 2, cdots`. 则有 `(ln_m x)' = prod_(0 le k lt m) ln_k^-1 x 又设 `M` 是某个使得 `ln_m n` 有定义的正整数. 则级数 `sum_(n ge M) ln_m^-p n (ln_m n)'` `= sum_(n ge M) 1/(n ln n ln_2 n cdots ln_(m-1) n ln_m^p n)` 在 `p gt 1` 时收敛, `p le 1` 时发散.
`ln_m n` 是增长十分缓慢的函数, 不过它仍趋向正无穷大. 本题表明以它的导函数为通项的级数恰好发散.

    简明起见只证 `m = 1` 情形. 一般情形是类似的.
  1. `p le 0` 时, 对充分大的 `n` 有 `1/(n ln^p n) ge 1/n`, 而调和级数发散; 由比较判别法一, 原级数发散.
  2. 设 `p = 1`. 对函数 `ln ln x` 在 `[n, n+1]` 上应用微分中值定理, 容易得到不等式 `ln ln(n+1) - ln ln n lt 1/(n ln n)`. 两边从 `2` 到 `n` 求和, `ln ln(n+1) - ln ln 2 lt sum_(k=2)^n 1/(k ln k)`. 从而原级数的部分和趋于无穷大, 因此是发散的.
  3. 设 `0 lt p lt 1`. 对函数 `ln^(1-p) x` 在 `[n, n+1]` 上应用微分中值定理, 得到不等式 `ln^(1-p)(n+1) - ln^(1-p) n lt (1-p)/(n ln^p n)`. 两边从 `1` 到 `n` 求和, `ln^(1-p)(n+1) lt sum_(k=1)^n (1-p)/(k ln^p k)`. 故原级数的部分和趋于无穷大, 因此发散.
  4. `p gt 1` 时, 同样对函数 `ln^(1-p) x` 在 `[n, n+1]` 上应用微分中值定理, 得到不等式 `(p-1)/((n+1)ln^p(n+1)) lt ln^(1-p) n - ln^(1-p)(n+1)`. 两边从 `2` 到 `n` 求和, `sum_(k=3)^n (p-1)/(k ln^p k) lt ln^(1-p)2 - ln^(1-p)(n+1)` `lt ln^(1-p)2`. 故原级数的部分和有上界, 因此收敛.

使用积分判别法, 本题是显然的 , 以 `p = 1` 为例, `sum_(n ge M) ln_m^-1 n (ln_m n)'` `= sum_(n ge M) (ln_(m+1) n)'` `= sum_(n ge M) int_n^(n+1) (ln_(m+1) n)' dx` `ge sum_(n ge M) int_n^(n+1) (ln_(m+1) x)' dx` `= int_M^oo (ln_(m+1) x)' dx = oo`.

等价无穷小 `(ln_m(n+1))/(ln_m n) - 1 ~ (ln_(m+1) n)'`, `n to oo`.

对 `m` 进行归纳, `m = 0` 时得 `(n+1)/n - 1 ~ 1/n`, 显然成立; 设结论对 `m-1` 成立, 考虑 `m` 的情形, `n to oo` 时有: `(ln_m(n+1))/(ln_m n) - 1` `= (ln_m(n+1) - ln_m n)/(ln_m n)` `= (ln{:(ln_(m-1)(n+1))/(ln_(m-1) n):})/(ln_m n)` `~ ((ln_m n)')/(ln_m n)` `= (ln_(m+1) n)'`.

    Gauss 检比法 将正项级数 `sum u_n` 的相邻两项比以 `f_m(n) = (ln_m n)'` 的阶渐近展开: `u_n//u_(n+1) = sum_(m=0)^N G_m f_m(n) + o(f_N(n))` `= G_0 + G_1/n + G_2/(n ln n) + G_3/(n ln n ln ln n) + cdots + o(f_N(n))`. 若存在 `M` 使得 `G_0 = G_1 = cdots = G_(M-1) = 1`, 而 `G_M != 1`, 则
  1. `G_M gt 1` (含 `G_M = +oo`) 时, 原级数收敛;
  2. `G_M lt 1` (含 `G_M = -oo`) 时, 原级数发散.
  1. 设 `G_M gt 1`, 取 `1 lt p lt G_M`, 则级数 `sum v_n = sum ln_(M-1)^-p n (ln_(M-1) n)'` 收敛. 考察它的前后项比, 利用引理将乘积展开: `v_n // v_(n+1)` `= (n+1)/n (ln(n+1))/(ln n) cdots (ln_(M-2) (n+1))/(ln_(M-2) n) ((ln_(M-1) (n+1))/(ln_(M-1) n))^p` `= (1 + f_M(n) + o(f_M(n)))^p prod_(m=1)^(M-1) (1+f_m(n) + o(f_m(n)))` 注意 `p lt G_M`, 故 `n` 充分大时, 上式小于等于 `u_n // u_(n+1)` `= 1 + 1/n + 1/(n ln n) + cdots + G_M f_M(n) + o(f_M(n))`. 由第三比较判别法知 `sum u_n` 收敛.
  2. `G_m lt 1` 的证明类似.

系数 `G_m` 可以通过计算极限 `lim_(n to oo) r_n^((m))` 得到, 其中 `r_n^((0)) = u_n // u_(n+1)`, `quad r_n^((m+1)) = (r_n^((m)) - G_m) ln_m n`.

积分判别法 设 `f` 在 `[M, +oo)` 上单调减, 于是 `f(n) = int_n^(n+1) f(n) dx ge int_n^(n+1) f(x) dx`, 对 `n` 求和得到 `sum_(n ge M) f(n) ge int_M^oo f(x) dx`, 这是级数的下界; 类似可得到上界. 于是无穷级数的敛散性归结为无穷积分的敛散性.

任意项级数

级数的两边夹法则 若级数 `sum_(n=1)^oo w_n` 与 `sum_(n=1)^oo v_n` 均收敛, 且对任意正整数 `n` 成立 `w_n le u_n le v_n`, 则 `sum_(n=1)^oo u_n` 收敛. 进一步, 若两端的级数收敛到同一个和, 则 `sum_(n=1)^oo u_n` 也收敛到这个和.

由 `w_n le u_n le v_n` 得 `0 le u_n - w_n le v_n - w_n`, 由已知 `sum_(n=1)^oo (v_n - w_n)` 收敛, 由正项级数的比较判别法知, `sum_(n=1)^oo (u_n - w_n)` 也收敛. 从而 `sum_(n=1)^oo u_n` 收敛. 进一步, 若两端级数收敛到同一个和, 则 `sum_(n=1)^oo (v_n - w_n)` 收敛到零, 从而 `sum_(n=1)^oo (u_n - w_n)` 也收敛到零, 即 `sum_(n=1)^oo u_n` 收敛到同一个和.

数列的两边夹法则要求两端的数列具有同一极限, 级数的两边夹法则无此要求.

Leibniz 判别法 如果数列 `{u_n}` 单调趋于零, 那么交错级数 `sum_(n=0)^oo (-1)^n u_n` 收敛.

Abel 变换 设 `a_1, a_2, cdots, a_m` 和 `b_1, b_2, cdots b_m` 是两组数. 记 `b_0 = 0`, `Delta b_k = b_k - b_(k-1)`, `k = 1, 2, cdots m`. 则 `sum_(k=1)^m a_k Delta b_k = a_k b_k - sum_(k=1)^(m-1) Delta a_(k+1) b_k`.

  1. Dirichlet 判别法 如果数列 `{u_n}` 单调趋于零, 级数 `sum_(n=1)^oo v_n` 的部分和有界, 那么 `sum_(n=1)^oo u_n v_n` 收敛.
  2. Abel 判别法 如果数列 `{u_n}` 单调有极限, 级数 `sum_(n=1)^oo v_n` 收敛, 那么 `sum_(n=1)^oo u_n v_n` 收敛.

Dirichlet 判别法把 Leibniz 判别法中的 `(-1)^n` 项 换成了更一般的部分和有界的 `v_n`, 是 Leibniz 判别法的推广.

反例

`0 le u_n le 1/n`, 而 `sum_(n=1)^oo (-1)^n u_n` 发散的例子: `u_n = (1+(-1)^n)/(2n)`. 在这个例子中, 通项 `u_n to 0`, 但不是单调的, 不满足 Leibniz 判别法的条件.
