第一类曲线积分 `int_L f(x,y,z) "d"s` 第二类曲线积分 `int_L bm F(x,y,z) * "d"bm r`
第一类曲面积分 `int_Sigma f(x,y,z) "d"sigma` 第二类曲面积分 `int_Sigma bm F(x,y,z) * "d"bm S`

形式上 切向微元 `"d"bm r = (dx, dy, dz)`, `quad "d"s = |"d"bm r|`,
法向微元 `"d"bm S = (dy dz, dz dx, dx dy)` `= (cos alpha, cos beta, cos gamma) "d"sigma`, `quad "d"sigma = |"d"bm S|`, 实际计算时, 为曲线 `L` (曲面 `Sigma`) 选取合适的参数, 就有 `"d"bm r = bm v dt`, `quad "d"s = |bm v| dt`;
`"d"bm S = bm n "d"u"d"v`, `quad "d"sigma = |bm n|"d"u"d"v`. 其中 切向量 `bm v = (dx/dt, dy/dt, dz/dt)`, `quad` 法向量 `bm n = |bm i, bm j, bm k; x_u, y_u, z_u; x_v, y_v, z_v|` 依赖于参数选取.

求 `int_L x y "d"s`, 其中 `L: {x+y+z = 0; x^2+y^2+z^2=1:}`

利用积分区域的对称性, `int_L x y "d"s` `= 1/3 int_L (x y+y z+z x) "d"s` `= 1/6 int_L [(x+y+z)^2 - (x^2+y^2+z^2)] "d"s` `= -1/6 int_L "d"s` `= -pi/3`.
