线性函数与双线性函数

线性函数

令 `V` 为 `bbb P` 上一线性空间, 视 `bbb P` 为一维线性空间, 称线性映射 `f: V to bbb P` 为 `V` 上一线性函数. `V` 上全体函数记为 `L(V, bbb P)`.

记 `bm alpha = (a_1, a_2, cdots, a_n)'`, `bm X = (x_1, x_2, cdots, x_n)'`, 则 `f(bm X) = alpha' X` 是 `bbb P^n` 上的一个线性函数.
方阵的迹是 `bbb P^(n xx n)` 上的一个线性函数.
多项式在一点的取值是 `bbb P[x]` 上的一个线性函数.

令 `V` 为 `bbb P` 上一线性空间, `"I" = (bm epsi_1, bm epsi_2, cdots, bm epsi_n)` 是一个基, `f` 是 `V` 上一个线性函数. 称 `bm epsi = (f(bm epsi_1), f(bm epsi_2), cdots, f(bm epsi_n))` 为 `f` 在基底 `"I"` 下的表示向量. 给定了表示向量, 任意 `bm alpha = sum_(i=1)^n x_i bm epsi_i in V` 的函数值都能确定了: 记 `bm X = (x_1, x_2, cdots, x_n)'`, 则 `f(bm alpha) = sum_(i=1)^n x_i f(bm epsi_i)` `= bm(epsi X)`.

双线性函数

令 `V` 为 `bbb P` 上一线性空间, 称映射 `f: V xx V to bbb P` 为 `V` 上一双线性函数, 如果它对每一个变元都是线性的: 即对任意 `bm alpha, bm beta, bm gamma in V` 和任意 `k, l in bbb P`, `(k bm alpha + l bm beta, bm gamma) = k(bm alpha, bm gamma) + l(bm beta, bm gamma)`,
`(bm gamma, k bm alpha + l bm beta) = k(bm gamma, bm alpha) + l(bm gamma, bm beta)`. `V` 上全体双线性函数记为 `BL(V, bbb P)`.
称 `f` 为对称的, 如果恒有 `f(bm alpha, bm beta) = f(bm beta, bm alpha)`; 称 `f` 为非奇异的, 如果对任意非零向量 `bm alpha in V`, 存在 `bm beta in V` 使得 `f(bm alpha, bm beta) != 0`; 称 `f` 为正定的, 如果对任意向量 `bm alpha in V`, `f(bm alpha, bm alpha) ge 0`, 等号成立当且仅当 `bm alpha = bm theta`. 显然若 `f` 正定, 则 `f` 非奇异.

令 `bm A in bbb P^(n xx n)`, `bm X = (x_1, x_2, cdots, x_n)'`, `bm Y = (y_1, y_2, cdots, y_n)'`, 则 `f(bm X, bm Y) = bm X' bm A bm Y = sum_(i,j=1)^n a_(i j) x_i y_j` 为 `bbb P^n` 上一双线性函数. 当 `bm A` 分别为对称的, 非奇异的和正定的时候, `f` 也分别为对称的, 非奇异的和正定的.

令 `V` 为 `bbb P` 上一线性空间, `"I" = (bm epsi_1, bm epsi_2, cdots, bm epsi_n)` 是一个基, `f` 是 `V` 上一个双线性函数. 称 `bm A = (f(bm epsi_i, bm epsi_j))_(n xx n)` 为 `f` 在基底 `"I"` 下的表示矩阵或度量矩阵. 给定了度量矩阵, 对任意 `bm alpha = sum_(i=1)^n x_i bm epsi_i` `= (bm epsi_1, bm epsi_2, cdots, bm epsi_n) bm X`,
`bm beta = sum_(i=1)^n y_i bm epsi_i` `= (bm epsi_1, bm epsi_2, cdots, bm epsi_n) bm Y`, 其函数值都能确定了: `f(bm alpha, bm beta) = sum_(i,j=1)^n f(bm epsi_i,bm epsi_j) x_i y_j` `= bm X' bm (A Y)`.

表示向量/度量矩阵与坐标变换 设 `bbb P` 上线性空间的两个基 `"I" = (bm eta_1, bm eta_2, cdots, bm eta_n)`, `"II" = (bm xi_1, bm xi_2, cdots, bm xi_n)` 之间的关系为 `(bm eta_1, bm eta_2, cdots, bm eta_n)` `= (bm xi_1, bm xi_2, cdots, bm xi_n) bm T`, 线性函数 `f` 在 `"I"` 下的表示向量为 `bm epsi`, 则它在 `"II"` 下的表示向量为 `bm epsi bm T`. 双线性函数 `g` 在 `"I"` 下的度量矩阵为 `bm A`, 则它在 `"II"` 下的度量矩阵为 `bm (T'A T)`, 即双线性函数在不同基底下的度量矩阵是合同的. 注意 `bm T` 可逆, 因而 `bm A` 和 `bm(T'A T)` 的秩相同, 称为 `f` 的秩.

仅验证度量矩阵的变换. 设 `bm alpha, bm beta` 在 `"I"` 下的坐标分别为 `bm X, bm Y`, 在 `"II"` 下的坐标分别为 `bm X_1, bm Y_1`, 则 `g(bm alpha, bm beta)` `= bm X' bm (A Y)` `= bm(T X_1)' bm A bm(T Y_1)` `= bm(X_1' T' A T Y_1)`. 可见 `g` 在 `"II"` 下的度量矩阵为 `bm(T'A T)`.

    设 `f` 为线性空间 `V` 上的双线性函数, 由于对称矩阵的合同仍为对称矩阵, 我们有以下各款等价:
  1. `f` 对称;
  2. `f` 在任一基底下的度量矩阵对称;
  3. `f` 在某一基底下的度量矩阵对称;
    设 `f` 为线性空间 `V` 上的双线性函数, 则以下各款等价:
  1. `f` 非奇异;
  2. `f` 在 `V` 的任意基底下的度量矩阵非奇异;
  3. `f` 在 `V` 的某一基底下的度量矩阵非奇异;
  4. `AA bm alpha in V\\{bm theta}`, `EE bm beta in V`, `f(bm alpha, bm beta) != 0`.
  1. `iff` 2. 取 `V` 的基底 `(bm epsi_1, bm epsi_2, cdots, bm epsi_n)`, 设 `f` 在该基底下的度量矩阵为 `bm A`, 则 `f` 非奇异 `iff bm alpha in V`, 关于 `bm beta` 的方程 `f(bm alpha, bm beta) = 0` 只有零解 `iff` 关于 `bm beta` 的方程 `f(bm epsi_i, bm beta) = 0`, `i = 1, 2, cdots, n` `iff` 关于 `bm Y` 的方程 `bm (A Y) = bb 0` 只有零解 `iff bm A` 非奇异. 类似可证 1. `iff` 4.
  2. `iff` 3. 只需注意到非奇异矩阵的合同矩阵仍非奇异.

对称双线性函数, 正交基底

设 `f` 为线性空间 `V` 上的对称双线性函数, 如果 `f(bm alpha, bm beta) = 0`, 则称 `bm alpha, bm beta` 正交. 如果 `V` 的一个基底中的不同向量两两正交, 则称为正交基底. 正交基底下的度量矩阵为对角阵, 对角线上非零元的个数恰为 `f` 的秩.

设 `f` 是秩为 `r` 的对称双线性函数, `f` 在正交基底 `(bm epsi_1, bm epsi_2, cdots, bm epsi_n)` 下的度量矩阵为 `bm D_(d_i)`, 不妨设 `f(bm epsi_i, bm epsi_i) != 0`, `i = 1, 2, cdots, r`. 令 `bm alpha = sum_(i=1)^n x_i bm epsi_i`, `bm beta = sum_(i=1)^n y_i bm epsi_i`, 我们有 `f(bm alpha, bm beta) = sum_(i=1)^r d_i x_i y_i`. 在复线性空间中, 若 `f(bm epsi_i, bm epsi_i) = d_i != 0`, 则 `f(bm epsi_i/sqrt(d_i), bm epsi_i/sqrt(d_i)) = 1`, 从而在适当的基底下有 `f(bm alpha, bm beta) = sum_(i=1)^r x_i' y_i'`. 在实线性空间中, 若 `d_i != 0`, 则 `f(bm epsi_i/sqrt(|d_i|), bm epsi_i/sqrt(|d_i|)) = sgn(d_i)`, 从而在适当的基底下有 `f(bm alpha, bm beta) = sum_(i=1)^p x_i'' y_i'' - sum_(i=p+1)^r x_i'' y_i''`.

Schmidt 正交化 设 `f` 为 `n` 维线性空间 `V` 上的对称双线性函数, 则存在 `V` 的正交基底. 换言之, 任意对称矩阵合同于对角形矩阵.

若 `f` 为零函数, 则 `V` 的任何基底都是正交基底. 现在设 `f` 是非零函数, 则 `EE bm alpha, bm beta in V`, `0 != f(bm alpha, bm beta)` `= 1/2 [ f(bm(alpha+beta), bm(alpha+beta)) - f(bm alpha, bm alpha) - f(bm beta, bm beta) ]`. 从而 `bm(alpha+beta)`, `bm alpha`, `bm beta` 三者不全为零, 即 `(EE bm epsi_1 in V)` `f(bm epsi_1, bm epsi_1) != 0`. 对 `n` 作归纳, 当 `n = 1` 时, 任何基底都是正交基底. 假定结论对所有维数小于 `n` 的线性空间都成立, 我们将 `bm epsi_1` 扩充为 `V` 的基底 `(bm epsi_1, bm eta_2, cdots, bm eta_n)`, 令 `bm epsi_i' = bm eta_i - (f(bm epsi_1, bm eta_i))/ (f(bm epsi_1, bm epsi_1)) bm epsi_1`, `quad i = 2, 3, cdots, n`. 设 `bm theta = k_1 bm epsi_1 + sum_(i=2)^n k_i bm epsi_i'` `= (k_1-c) bm epsi_1 + sum_(i=2)^n k_i bm eta_i`, 容易推出 `bm epsi_1, bm epsi_2', cdots, bm epsi_n'` 线性无关. 又 `f(bm epsi_1, bm epsi_i')` `= f(bm epsi_1, bm eta_i - (f(bm epsi_1, bm eta_i))/ (f(bm epsi_1, bm epsi_1)) bm epsi_1)` `= 0`,
`i = 2, 3, cdots, n`. 因此对任意 `bm alpha in W := G[bm epsi_2', cdots, bm epsi_n']`, `f(bm epsi_1, bm alpha) = 0`. 由归纳假设, `W` 中存在正交基底 `(bm epsi_2, cdots, bm epsi_n)` 但 `bm epsi_1` 与 `W` 中任意向量都正交, 因此 `(bm epsi_1, bm epsi_2, cdots, bm epsi_n)` 是 `V` 的一个正交基底.

非奇异对称双线性函数, 子空间的正交补

令 `f` 为线性空间 `V` 上的非奇异对称双线性函数. 则 `(V, f)` 称为一个对称双线性度量空间, 仍简记为 `V`. 特别 `V` 为一实线性空间时, 称为伪 Euclid 空间.

对称双线性度量空间存在正交基底 `(bm epsi_1, bm epsi_2, cdots, bm epsi_n)`. 因为 `f` 非奇异, 有 `f(bm epsi_i,bm epsi_j) { = 0, if i != j; != 0, if i = j; :}`

令 `(V, f)` 为一对称双线性度量空间, `W le V`, 则 `V` 中满足与 `W` 中任意向量都正交的向量的全体构成 `V` 的子空间: `W^_|_ := {bm alpha in V | (AA bm beta in W) f(bm alpha, bm beta) = 0} le V`, 称为 `W` 的正交补空间.

  1. 令 `(V, f)` 为一对称双线性度量空间, `W le V`, `"dim"W = r ge 1`, 取 `W` 的基底 `(bm alpha_1, bm alpha_2, cdots, bm alpha_r)`, 则 `W^_|_ = {bm x in V | f(bm alpha_i, bm x) = 0, i = 1, 2, cdots, r}`, 即 `W^_|_` 是关于 `bm x in V` 的方程组 `f(bm alpha_i, bm x) = 0`, `quad i = 1, 2, cdots, r` 的解空间. 取 `V` 的基底 `(bm epsi_1, bm epsi_2, cdots, bm epsi_n)`, 设 `f` 在该基底下的度量矩阵为 `bm T` (显然 `bm T` 对称非奇异), `(bm alpha_1, bm alpha_2, cdots, bm alpha_r)` `= (bm epsi_1, bm epsi_2, cdots, bm epsi_n) bm A`,
    `bm x = (bm epsi_1, bm epsi_2, cdots, bm epsi_n) bm X`,
     则方程组可以表示为 `bm (A'T X) = bb 0`. 特别 `bm T = bm E` 时, 有 `f(bm x, bm y) = sum_(i=1)^n x_i y_i`, 方程组简化为 `bm(A'X) = bb 0`.
  2. 现在考虑齐次线性方程组 `bm(A'X) = bb 0`, 记 `bm A = (bm A_1, bm A_2, cdots, bm A_r)`, 取 `bbb P^n` 上非奇异对称双线性函数 `f_(bm E)`, 使得其在自然基底 `(bm epsi_1, bm epsi_2, cdots, bm epsi_n)` 下的度量矩阵为 `bm E`. 则 `f_(bm E)(bm X, bm Y) = bm(X'Y)`. 齐次线性方程组 `bm(A'X) = bb 0` 写为双线性函数的形式即 `f_(bm E)(bm A_i, bm X) = 0`, `quad i = 1, 2, cdots, r`. 因此这个方程组的解空间是 `G[bm A_1, bm A_2, cdots, bm A_r]^_|_`.
    令 `(V,f)` 为一对称双线性度量空间, `W le V`, 则
  1. `"dim" W + "dim" W^_|_ = "dim" V`;
  2. `(W^_|_)^_|_ = W`.
    记 `"dim"V = n`, `"dim"W = r`. 由 `f` 非奇异知 `V^_|_ = {bm theta}`, `{bm theta}^_|_ = V`, 所以结论 1, 2 成立, 下设 `r ge 1`.
  1. 假设如 的 1, 由于 `bm alpha_i` 线性无关, 从而 `bm A` 的列向量线性无关, `r_(bm A) = r`. 由 `bm T` 非奇异知, 方程组 `bm(A'T X) = bb 0` 的系数矩阵的秩 `r_(bm(A'T)) = r`, 从而方程组的解空间维数为 `n-r`. 又显然该方程组的解空间同构于 `W`, 所以 `"dim"W^_|_ = n-r`.
  2. 由 1. 有 `"dim"W^_|_ = n-r`, `"dim"(W^_|_)^_|_ = r`. 又 `W sube (W^_|_)^_|_` `= {bm alpha in V | (bm beta in W^_|_) f(bm alpha, bm beta) = 0}`, `V` 的两个子空间 `W` 与 `(W^_|_)^_|_` 的维数相同, 又有包含关系, 从而 `W` 的基也是 `(W^_|_)^_|_` 的基. 所以它们是同一个子空间.

尽管有维数公式 `"dim"W + "dim"W^_|_ = "dim"V`, 但未必有 `V = W o+ W^_|_`. 如 `CC^2` 上的非奇异对称双线性函数 `f((x_1","y_1)","(x_2","y_2)) = x_1 x_2 + y_1 y_2`, 取 `W = G[(1","i)]`, 则 `W^_|_ = W`. 这暗示我们, `f(bm X, bm Y) = sum_(i=1)^n x_i y_i` 不是复线性空间中合适的双线性函数, 而 `f(bm X, bm Y) = sum_(i=1)^n bar x_i y_i` 更合适一些. 事实上, 后者是一个内积.

设 `bm A in bbb P^(m xx n)`, `bm X in bbb P^n`, `bm B, bm Y in bbb P^m`, 考虑非齐次方程组 `bm(A X) = bm B` 的转置齐次线性方程组 `bm(A' Y) = bb 0`. 在 `(bbb P^m, f_(bm E))` 中, 后者的解空间是 `W := G[bm A_1, bm A_2, cdots, bm A_n]^_|_`, 则 `bm(A X) = bm B` 有解当且仅当 `bm B in G[bm A_1, bm A_2, cdots, bm A_n] = W^_|_`.

内积空间

内积的定义

  1. 令 `V` 为一实线性空间, 称 `V` 上对称正定的双线性函数 `(*,*) : V xx V to RR` 为 `V` 上一内积. `V` 连同内积 `(*,*)` 构成的代数系统称为一实内积空间, 或Euclid 空间.
  2. 设 `V` 为一复线性空间, `V` 上的二元函数 `(*,*): V xx V to CC` 称为 `V` 上一内积, 如果满足
    1. 正定性. `(AA bm x in V)` `(bm x, bm x) ge 0`, 等号成立当且仅当 `bm x = 0`;
    2. 共轭对称性. `(AA bm x, bm y in V)` `(bm x, bm y) = bar((bm y","bm x))`;
    3. 关于第一变元的线性性和第二变元的共轭线性性. `(AA bm x, bm y, bm z in V)` `(AA k, l in bbb F)` `(k bm x+l bm y, bm z) = k(bm x, bm z) + l(bm y, bm z)`,
      `(bm z, k bm x+l bm y) = bar k(bm z, bm x)+bar l(bm z, bm y)`.
    `V` 连同内积 `(*,*)` 构成的代数系统称为一复内积空间.

复线性空间中内积关于第二变元的共轭线性性可以由其它性质推出.

  1. `RR^n` 上的二元函数 `(bm X, bm Y) = sum_(i=1)^n x_i y_i = bm(X'Y)` 是一个内积.
  2. `CC^n` 上的二元函数 `(bm X, bm Y) = sum_(i=1)^n bar x_i y_i = bm(bar X'Y)` 是一个内积.
  3. `C[a,b]` 上的二元函数 `(f(x),g(x)) = int_a^b f(x) g(x) dx` 是一个内积.
  4. 规定两个函数相等是在几乎处处意义下的, 则 `L^2[a,b]` 上的二元函数 `(f(x),g(x)) = int_a^b f(x) g(x) dx` 是一个内积.

Cauchy-Schwarz 不等式

令 `V` 为 `bbb P` 上一内积空间 (`bbb P = CC` 或 `RR`), 则 `|(bm x","bm y)|^2 le (bm x, bm x)(bm y, bm y)`, `quad AA bm x, bm y in V`. 等号成立当且仅当 `bm x, bm y` 线性相关.

当 `bm y = bm theta` 时, 不等式取得等号, 显然成立; 否则取 `t = -((bm x","bm y))/((bm y","bm y))`, 考虑向量 `bm x + t bm y`, 由正定性有 `0 le (bm x + t bm y, bm x + t bm y)` `= (bm x, bm x) + bar t(bm x, bm y) + t bar((bm x","bm y)) + t bar t(bm y, bm y)` `= (bm x, bm x) + 2 Re(bar t (bm x","bm y)) + |t|^2(bm y, bm y)` `= (bm x, bm x) -2 |(bm x","bm y)|^2/((bm y","bm y)) + |(bm x","bm y)|^2/((bm y","bm y))` `= (bm x, bm x) - |(bm x","bm y)|^2/((bm y","bm y))`. 变形得 `|(bm x","bm y)|^2 le (bm x, bm x)(bm y, bm y)`. 现在考虑等号成立的条件. 当 `bm x, bm y` 线性相关时, `bm y = bm theta` 的情形已经考虑过. 现在设 `bm x = lambda bm y`, 代入有 `(lambda bm y, lambda bm y)(bm y, bm y)` `= |lambda|^2 |(bm y","bm y)|^2` `= |(lambda bm y","bm y)|^2`, 故等号成立. 反之设不等式中的等号成立, 不妨设 `bm y != bm theta`, 由内积的正定性, 这等号成立当且仅当 `bm x + t bm y = bm theta`, 即 `bm x, bm y` 线性相关.

    将 Cauchy-Schwarz 不等式应用到具体的内积空间中, 有
  1. 对任意实数 `x_i, y_i`, `i = 1, 2, cdots, n`, 有 `|sum_(i=1)^n x_i y_i| le sqrt(sum_(i=1)^n x_i^2) sqrt(sum_(i=1)^n y_i^2)`.
  2. 对任意 `f(x), g(x) in C[a,b]`, 有 `|int_a^b f(x)g(x) dx| le sqrt(int_a^b f^2(x) dx) sqrt(int_a^b g^2(x) dx)`.

线性赋范空间与度量空间

我们将在内积空间中引入向量的长度, 距离, 夹角的概念.

    设 `V` 为 `bbb P` 上的线性空间, 实函数 `||*||: V to RR` 称为 `V` 上的范数, 如果满足
  1. 正定性. `(AA bm x in V)` `||bm x|| ge 0`, 等号成立当且仅当 `bm X = bm theta`.
  2. `(AA bm x in V)` `(AA c in bbb P)` `||c bm x|| = |c|` `||bm x||`.
  3. 次可加性, 或三角不等式. `(AA bm x, bm y in V)` `||bm x+bm y|| le ||bm x|| + ||bm y||`.
    4. `V` 连同范数 `||*||` 构成的代数系统称为一线性赋范空间. 范数的直观意义是几何向量的长度.

在内积空间 `V` 中引入 `|bm x| = sqrt((bm x","bm x))`, 则 `|*|` 为一范数.

  1. `|*|` 的正定性由内积的正定性保证.
  2. 容易验证.
  3. 计算 `|bm x+bm y|^2` `= (bm x+bm y,bm x+bm y)` `= (bm x,bm x) + 2 Re(bm x, bm y) + (bm y, bm y)` `le (bm x, bm x) + |(bm x","bm y)| + (bm y, bm y)` `le |bm x|^2 + |bm x|` `|bm y| + |bm y|^2` `= (|bm x|+|bm y|)^2`, 所以三角不等式成立.
    设 `X` 为一集合, 二元实函数 `d: X xx X to RR` 称为 `X` 上的距离函数或度量函数, 如果满足
  1. 正定性. `(AA x, y in X)` `d(x,y) ge 0`, 等号成立当且仅当 `x=y`.
  2. 对称性. `(AA x, y in X)` `d(x,y) = d(y,x)`.
  3. 三角不等式. `(AA x,y,z in X)` `d(x,y) le d(x,z) + d(z,y)`.
    4. `(X,d)` 称为度量空间.

在线性赋范空间中, 规定 `d(x,y) = ||x-y||`, 可以验证 `d` 是一个距离函数.

令 `V` 为内积空间, 规定两个非零向量 `bm x, bm y` 的夹角为 `(:bm x, bm y:) = arccos{:((bm x","bm y))/(|bm x| |bm y|):}`. 由 Cauchy-Schwarz 不等式知, 上述定义有意义, 且夹角的可能范围是 `[0, pi]`.

勾股定理 令 `V` 为内积空间, `bm x, bm y` 正交, 则 `|bm x+bm y|^2 = |bm x|^2 + |bm y|^2`. 勾股定理可以推广到多个两两正交的向量.

由 `(bm x, bm y) = 0` 有 `|bm x+bm y|^2 = (bm x+bm y,bm x+bm y)` `= (bm x,bm x) + 2 Re(bm x, bm y) + (bm y, bm y)` `= |bm x|^2 + |bm y|^2`.

内积空间的标准正交基底

`bm beta_1 = bm alpha_1`,
`bm beta_i = bm alpha_i - sum_(j=1)^(i-1) ((bm alpha_i","bm beta_j)) /((bm beta_j","bm beta_j)) bm beta_j`, `quad i = 2, 3, cdots, n`.
    设 `f` 为实线性空间 `V` 上的双线性函数, 则以下各款等价:
  1. `f` 对称正定 (根据下文的定义, 即 `f` 是一个内积);
  2. `f` 在 `V` 的任意基底下的度量矩阵正定;
  3. `f` 在 `V` 的某一基底下的度量矩阵正定.

向量到子空间的距离, 最小二乘法

最小二乘法, 好比在平面上找一点, 使它到空间中一点距离最小. 只不过这里的 "平面", "空间" 都指线性空间.

线性回归 假设有数据点 `(x_1, y_1), (x_2, y_2), cdots, (x_n, y_n)`. 在平面上用一直线 `y = a x + b` 拟合这些数据, 使得每个点处误差的平方和 `E(a, b) = sum_(i=1)^n (y_i - a x_i - b)^2` 最小.

令偏导数等于零, `0 = (del E)/(del b) = -2 sum_(i=1)^n (y_i - a x_i - b)`,
`0 = (del E)/(del a) = -2 sum_(i=1)^n x_i (y_i - a x_i - b)`. 即 `sum y_i = a sum x_i + n b`,
`sum x_i y_i = a sum x_i^2 + b sum x_i` 于是 `b = bar y - a bar x`, `quad a = (sum x_i y_i - n bar x_i bar y_i)/(sum x_i^2 - n {:bar x:}^2)`.

用线性代数的语言, 这个问题就是: 设 `bm 1 = (1, 1, cdots, 1) in RR^n`, `bm x = (x_1, x_2, cdots, x_n) in RR^n`, `bm y = (y_1, y_2, cdots, y_n) in RR^n`, 内积定义为 `(bm x, bm y) = sum_(i=1)^n x_i y_i`. 在 `RR^n` 的子空间 `"span"[1, bm x]` 中寻找一向量, 使其到向量 `bm y` 的距离最小. 这一向量的坐标 `(b, a)` 满足 `[(bm 1, bm 1), (bm 1, bm x); (bm x, bm 1), (bm x, bm x)] [b;a] = [(bm 1, bm y); (bm x, bm y)]`, 即 `[sum 1, sum x_i; sum x_i, sum x_i^2] [b;a] = [sum y_i; sum x_i y_i]`. 解得 `b = bar y - a bar x`, `quad a = (sum x_i y_i - n bar x_i bar y_i)/(sum x_i^2 - n {:bar x:}^2)`.
