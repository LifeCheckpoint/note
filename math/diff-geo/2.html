正则曲线: 无穷可微, 不停留

  1. 每一分量是 `C^oo` 函数;
  2. `|("d" bm r)/dt| gt 0`, `AA t in (a, b)`.

以后我们提到曲线, 如无特别说明, 都指正则曲线.

`bm r(t) = (x(t), y(t))`, `t in (a, b)`

法向量 `(dy/dt, -dx/dt)`
单位正法向量 `bm n = (-dy/("d"s), dx/("d"s))`
Frenet 标架 `{bm r";" bm t, bm n}`

记号 `bm n` 的含义与参考链接中的不同, 这里表示单位正法向量, 它是与 `bm t` 正交的单位向量, 且 `{bm t, bm n}` 成右手系.

曲率

因为 `bm t` 的长度不变, 所以 `(:bm t, bm t':) = 0`. 但 `(:bm t, bm n:) = 0`, 所以 `bm t'` 与 `bm n` 线性相关. 设 `("d" bm t)/("d"s) = kappa(s) bm n`. 称 `kappa(s)` 为曲率. 显然 `|kappa(s)| = |("d" bm t)/("d"s)|`.

Frenet 方程

对 `(:bm t, bm n:) = 0` 求导知 `(:bm t', bm n:) + (:bm t, bm n') = 0`, 故 `"d"/("d"s) [bm t; bm n] = [0, kappa; -kappa, 0][(bm t),(bm n)]`. `kappa gt 0` 时, `bm n` 沿逆时针转动 (曲线沿逆时针绕行).

三维正则曲线

`bm r(t) = (x(t), y(t), z(t))`

相关记号

位移矢量微元 `"d" bm r = (dx, dy, dz)`
单位切向量 `bm t = ("d" bm r)/("d"s)`
曲率向量 `("d" bm t)/("d"s) = (d^2 bm r)/{:"d"s:}^2`
曲率 `kappa(s) = |("d" bm t)/("d"s)|`
主法向量 `bm n = 1/kappa ("d" bm t)/("d"s)`
副法向量 `bm b = bm t ^^ bm n`
挠率 `tau = (:("d" bm n)/("d"s), bm b:)`
Frenet 标架 `{bm r";" bm t, bm n, bm b}`

`{bm t, bm n, bm b}` 形成右手系, 且均为单位向量. 以这三个向量为法向量的平面分别称为法平面, 从切平面和密切平面.

Frenet 方程

设 `"d"/("d"s) [bm t; bm n; bm b] = (omega)_(ij) [bm t; bm n; bm b]`, 由于 `bm t, bm n, bm b` 是两两正交的单位向量, 所以 `omega_(ij) = -omega_(ji)`. 又由主法向量的定义知 `omega_(12) = kappa`, `omega_(13) = 0`. 设 `omega_(23) = (:("d" bm n)/("d"s), bm b:) = tau`, 综上得到 `"d"/("d"s) [bm t; bm n; bm b] = [ 0, kappa, 0; -kappa, 0, tau; 0, -tau, 0; ][bm t; bm n; bm b]`.

不同于平面曲线, 空间曲线总有 `kappa ge 0`. `tau gt 0` 时, 曲线沿正向穿越密切平面 (即在 `(bm t, bm b)` 平面的投影是上升曲线).

直观上, 空间曲线的曲率可以理解为向心力, 挠率则可以理解为密切平面的运动速率.

`E^3` 的正则曲线 `bm r(t)` 的曲率和挠率分别为 (以下导数均表示对 `t` 求导) `kappa(t) = |bm r' ^^ bm r''|/|bm r'|^3`, `quad` `tau(t) = ("("bm r', bm r'', bm r'''")")/|bm r' ^^ bm r''|^2`.

`{:"d"s:}/dt = |bm r'|`, `quad` ` bm t = {:"d"bm r:}/{:"d"s:} = {:"d"bm r:}/dt dt/{:"d"s:} = (bm r')/|bm r'|` 应用例1-1, ` {"d"^2 bm r:}/{:"d"s:}^2 = "d"/dt ({:"d"bm r:}/{:"d"s:}) dt/{:"d"s:} = 1/|bm r'| "d"/dt ((bm r')/|bm r'|) = (bm r' ^^ (bm r'' ^^ bm r')) / |bm r'|^4`. 注意到 `(:bm r', bm r'' ^^ bm r':) = 0`, 所以 ` kappa(t) = |{:"d"^2 bm r:}/{:"d"s:}^2| = (|bm r'| |bm r'' ^^ bm r'|)/|bm r'|^4 = |bm r'' ^^ bm r'|/|bm r'|^3`,
` bm n = (bm r' ^^ (bm r'' ^^ bm r'))/(|bm r'| |bm r'' ^^ bm r'|)`,
` bm b = bm t ^^ bm n = (bm r' ^^ (bm r' ^^ (bm r'' ^^ bm r'))) / (|bm r'|^2 |bm r'' ^^ bm r'|) = ((:bm r',bm r'' ^^bm r':)bm r'-(:bm r',bm r':)(bm r'' ^^bm r')) / (|bm r'|^2 |bm r'' ^^ bm r'|)` `= -(bm r'' ^^ bm r')/|bm r'' ^^ bm r'|`. 记 `bm beta = bm r'' ^^ bm r'`, 所以 `bm beta' = bm r''' ^^ bm r'`,
` -bm beta' ^^ bm beta = bm beta ^^ (bm r''' ^^ bm r') = (:bm beta, bm r':) bm r''' - (:bm beta, bm r''':)bm r'` `= -(:bm beta, bm r''':)bm r' = (bm r', bm r'', bm r''') bm r'`. 利用已有结果, ` {:"d"bm b:}/{:"d"s:} = "d"/dt (-(bm beta)/|bm beta|) dt/{:"d"s:} = -(bm beta ^^ (bm beta' ^^ bm beta))/(|bm r'| |bm beta|^3) = ((bm r', bm r'', bm r''') bm beta ^^ bm r') / (|bm r'| |bm beta|^3)`. 联系 `bm n = (bm r ^^ bm beta)/(|bm r'| |bm beta|)` 和 `{:"d"bm b:}/{:"d"s:} = -tau(t) bm n` 即得到 `tau(t)` 的表达式.

设 `bm r(t) = (x(t), y(t))`, 下式各导数均表示对 t 求导, 则曲率 `kappa(t) = (x'y'' - x''y')/(x'^2 + y'^2)^(3//2)`.

自然, `kappa` 可以通过计算 `("d"^2 bm r)/{:"d"s:}^2` 与 `bm n` 关于 `x, y` 的表达式来求得. 不过也可以将二维问题视为三维问题, 利用上题结论, `kappa(t) = +- (x' y'' - x'' y')/(x'^2 + y'^2)^(3/2)`. 为确定其符号, 设 `kappa != 0`, 因为 `{bm t, bm n}` 为正定向的, 所以 `(bm t, bm n, bm k) gt 0`, `bm k` 是 `z` 轴的单位向量. 由于 `bm n = 1/kappa ("d"bm t)/{:"d"s:}`, 上式化为 `(bm r', bm r'', kappa bm k) gt 0`, 即 `|x', y', 0; x'', y'', 0; 0, 0, kappa| gt 0`. 可见前述式子应当取正号.

在极坐标下, 曲线 `r = f(theta)` 的曲率 `kappa(theta) = (r^2 + 2r'^2 - rr'')/(r^2 + r'^2)^(3//2)`. 其中各导数均表示对 `theta` 求导.

`{ x = r(theta) cos theta; y = r(theta) sin theta; :}` `{ x' = r' cos theta - r sin theta; y' = r' sin theta + r cos theta; :}` `{ x'' = r'' cos theta - r' sin theta - y'; y'' = r'' sin theta + r' cos theta + x'; :}` 易知 `x'^2 + y'^2 = r^2 + r'^2`, 将这些计算结果代入上一题, 有 `kappa = (r^2 + r'^2)^(-3/2)(x'^2 + x'(r'' sin theta + r' cos theta) - y'(r'' cos theta - r' sin theta) + y'^2)`, 即为所求.

圆柱螺旋线 `bm r(t) = (a cos t, a sin t, bt)` (`a gt 0`) 的曲率等于 `a/(a^2+b^2)`, 挠率等于 `b/(a^2+b^2)`.

平面曲线的曲率恒等于 0 当且仅当它是直线; 恒等于非零常数当且仅当它是半径为 `1/kappa` 的圆.

设空间曲线 `bm r` 的曲率 `kappa` 和挠率 `tau` 为常数, 则

  1. `kappa = 0`, `bm r` 是直线;
  2. `kappa gt 0`, `tau = 0`, `bm r` 是半径为 `1/kappa` 的圆;
  3. `kappa gt 0`, `tau != 0`, `bm r` 是半径为 `kappa/(kappa^2+tau^2)`, 速率为 `tau/(kappa^2+tau^2)` 圆柱螺旋线.

空间曲线的曲率恒不等于 0 时, 它落在一张平面上当且仅当其挠率恒等于 0.

设曲线 `bm r` 落在以 `bm a` 为单位法向量的平面上, 则 `bm r` 的切向量与 `bm a` 垂直, 即 `(:bm t, bm a:) = 0`. 求导得 `kappa (:bm n, bm a:) = 0`. 假设 `kappa(s) != 0`, 得到 `(:bm n, bm a:) = 0`. 再求导得 `(:-kappa bm t + tau bm b, bm a:) = 0`, 因此 `tau (:bm b, bm a:) = 0`. 由 `bm a` 与 `bm t, bm n` 垂直知 `bm a` 与 `bm b` 平行, 从而 `(:bm b, bm a:) = 1`, `tau -= 0`.
反之设 `tau -= 0`, 由 `("d"bm b)/{:"d"s"} = -tau bm n` 知 `bm b` 是常向量. 因为 `bm t` 与 `bm b` 垂直, `bm t` 只能落在同一平面内, 即曲线 `bm r` 落在同一平面内.

曲线论基本定理

曲线的弧长, 曲率和挠率在刚体运动 (同向合同变换) 下不变.

平面和空间的 Frenet 方程给出了各自 Frenet 标架的运动方程, 它们是一阶线性常微分方程组. 下面的两个定理说明, 我们可以通过求解此方程组来重建曲线, 且此曲线是唯一的. 换言之, 在相差一个刚体运动下, 曲线由曲率和挠率唯一决定.

设 `E^3` 中有两条弧长参数曲线 `bm r_1(s), bm r_2(s)` 定义在同一参数区间 `(a, b)` 上, 且它们的曲率与挠率在 `(a, b)` 上恒等: `kappa_1(s) = kappa_2(s) gt 0`, `tau_1(s) = tau_2(s)`, `AA s in (a, b)`. 则存在 `E^3` 的一个刚体运动使两条曲线相等.

设 `kappa(s), tau(s) in C^1(a, b)`, `kappa(s) gt 0`. 则存在 `E^3` 的弧长参数曲线, 它以 `s` 为弧长参数, 以 `kappa` 为曲率, `tau` 的挠率.
