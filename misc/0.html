[问题来源] 求 `(sum_(n=1)^oo n/((2n+1)!))/(sum_(n=1)^oo n/((2n-1)!))`.

将求和写成 `(2/(3!) + 4/(5!) + 6/(7!) + cdots)/ (2/(1!) + 4/(3!) + 6/(5!) + cdots)` 分子等于 `(3-1)/(3!) + (5-1)/(5!) + (7-1)/(7!) + cdots` `= 1/(2!) - 1/(3!) + 1/(4!) - 1/(5!) + 1/(6!) - 1/(7!) + cdots` `= "e"^-1 - 1 + 1/(1!)` `= "e"^-1`. 类似可得分母等于 `"e"`, 于是结果为 `"e"^-2`.

化简 `root 3(3+sqrt(275/27)) + root 3(3-sqrt(275/27)) = root 3 2`.

`lim_(n to oo) "e"^-n sum_(k=0)^n n^k/(k!) = 1/2`.

`lim_(n to oo) sum_(k=1)^n (k/n)^n = "e"/("e"-1)`.

`"d"/dx ln Gamma(x)` 的递推公式. `Gamma(x+1) = x Gamma(x)`
`ln Gamma(x+1) = ln x + ln Gamma(x)`
`"d"/dx ln Gamma(x+1) = 1/x + "d"/dx ln Gamma(x)`. 记 `f(x) = "d"/dx ln Gamma(x) = (Gamma'(x))/(Gamma(x))`, 则 `f(x+1) - f(x) = 1/x`. 证明: `(Gamma'(x))/(Gamma(x))|_(x=1) = int_0^oo "e"^-t ln t dt = -gamma`, `gamma = lim_(n to oo) sum_(k=1)^n 1/k - ln n` 是欧拉常数.
