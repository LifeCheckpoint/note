拓扑空间

拓扑空间的定义

    拓扑公理 设 `X != O/`, 称 `tau sube 2^X` 为 `X` 的一个拓扑, 如果它满足
  1. `X, O/ in tau`;
  2. `tau` 中任意多个集合的并仍在 `tau` 中;
  3. `tau` 中任意有限个集合的交仍在 `tau` 中.
    4. 集合 `X` 连同其上的拓扑 `tau` 组成一个拓扑空间 `(X, tau)`, 不引起混淆的情况下, 简记为 `X`. `tau` 中的成员称为这个拓扑空间的开集. 因此, 给出集合的一个拓扑就是指定它的哪些子集是开集 (在满足拓扑公理的前提下).

应用归纳法知, 拓扑公理的 3. 等价于 `tau` 中任意两个集合的交仍在 `tau` 中.

设 `tau_1, tau_2` 是 `X` 上的两个拓扑. 称 `tau_2` 比 `tau_1` 大或精细, 如果 `tau_1 sube tau_2`.

    设 `X != O/`.
  1. 平凡拓扑 `tau_t = {X, O/}` 是 `X` 上最小的拓扑, 它的开集只有全空间与空集;
  2. 离散拓扑 `tau_s = 2^X` 是 `X` 上最大的拓扑, `X` 的任一子集都是开集;
  3. 余有限拓扑 `tau_f = {A^c: A sube X, |A| lt oo} uu {O/}`, `X` 为无穷集合;
  4. 余可数拓扑 `tau_c = {A^c: A sube X, |A| = aleph_0} uu {O/}`, `X` 为不可数集合;
  5. 欧氏拓扑 `tau_e = {uuu_(alpha in I) O_alpha: O_alpha "是开区间"}`, `X = RR`. 这里 `I` 可以是空集, 因此 `O/ in tau_e`.
    6. 统一取 `X = RR` 时, `tau_f` 小于 `tau_c` 和 `tau_e`; 而 `tau_c, tau_e` 不能比较大小.

度量拓扑

    设 `X != O/`, 称映射 `d: X xx X to RR` 为 `X` 上的一个度量, 如果它满足
  1. 正定性. `(AA x in X)` `d(x, x) = 0`, `(AA x, y in X, x != y)` `d(x, y) gt 0`;
  2. 对称性. `(AA x, y in X)` `d(x, y) = d(y, x)`;
  3. 三角不等式. `(AA x, y, z in X)` `d(x, z) le d(x, y) + d(y, z)`.
    4. 集合 `X` 连同其上的度量 `d` 组成一个度量空间 `(X, d)`, 有时也简记为 `X`.

在 `RR^n` 上规定度量 `d(x, y) = (sum_(i=1)^n (x_i - y_i)^2)^(1/2)`, 称 `E^n = (RR^n, d)` 为 `n` 维欧氏空间.

度量空间 `(X, d)` 上以 `x_0` 为心, `epsi gt 0` 为半径的开球定义为 `B(x_0, epsi) := {x in X: d(x_0, x) lt epsi}`.

度量空间 `(X, d)` 的任意两个开球的交可以表为一族开球的并.

设 `U = B(x_1, epsi_1) nn B(x_2, epsi_2)`, 任取 `x in U`, 有 `d(x, x_i) lt epsi_i`, `i = 1, 2`. 取 `epsi_x = min{epsi_1 - d(x, x_1), epsi_2 - d(x, x_2)}`, 则对任意 `y in B(x, epsi_x)`, 有 `d(x, y) lt epsi_x le epsi_i - d(x, x_i)`, `i = 1, 2`. 于是 `d(x_i, y) le d(x_i, x) + d(x, y) lt epsi_i`, `i = 1, 2`, 即 `B(x, epsi_x) sube U`. 从而 `U = uuu_(x in U) B(x, epsi_x)`.

任意度量空间上都可以诱导出一个拓扑 `tau_d = {uuu_(alpha in I) O_alpha: O_alpha "是开球"}`, 称为度量拓扑. 特别把欧氏空间 `E^n` 的度量拓扑称为欧氏拓扑.

显然 `tau_d` 满足拓扑公理的 1 和 2 (注意 `I` 可以是空集). 下证它满足 3. 设 `U = uuu_alpha B(x_alpha, epsi_alpha), V = uuu_beta B(x_beta, epsi_beta) in tau_d`, 则 ` U nn V = (uuu_alpha B(x_alpha, epsi_alpha)) nn (uuu_beta B(x_beta, epsi_beta))` `= uuu_(alpha, beta) (B(x_alpha, epsi_alpha) nn B(x_beta, epsi_beta))`. 再由引理知 `U nn V in tau_d`.

拓扑空间中的几个基本概念

闭集

称拓扑空间 `X` 的子集 `A` 为闭集, 如果 `A^c` 是开集. `(X, tau)` 的全体闭集可以写作 `{A^c: A in tau}`.

  1. 平凡拓扑空间中只有两个闭集: `X` 与 `O/`;
  2. 离散拓扑空间中, 任何子集都是闭集;
  3. 余有限拓扑空间 `(X, tau_f)` 中, 闭集是 `X` 或有限集;
  4. 余可数拓扑空间 `(X, tau_c)` 中, 闭集是 `X` 或可数集.
    由拓扑公理和 De Morgan 公式, 拓扑空间的闭集满足:
  1. `X`, `O/` 都是闭集;
  2. 任意多个闭集的交仍是闭集;
  3. 有限个闭集的并仍是闭集.

邻域, 内点与内部

设 `X` 为拓扑空间, `A sube X`, `x in A`. 如果存在开集 `U` 满足 `x in U sube A`, 则称 `x` 是 `A` 的一个内点, `A` 是 `x` 的一个邻域. 特别地, 若 `A` 本身是开集, 则称 `A` 是 `x` 的开邻域. 由定义知道, `A` 是 `x` 的邻域当且仅当它含着 `x` 的一个开邻域. `A` 的全体内点的集合称为 `A` 的内部, 记为 `A^@` 或 `overset @ A` 或 `"int" A`.

  1. `A sube B rArr A^@ sube B^@`;
  2. `A^@` 是 `A` 中所有开集的并, 因此是 `A` 中最大的开集;
  3. `A` 是开集 `iff A^@ = A`;
  4. `(A nn B)^@ = A^@ nn B^@`;
  5. `(A uu B)^@ supe A^@ uu B^@`.
  1. `x in A^@` `iff EE U in tau, x in U sube A` `rArr EE U in tau, x in U sube B` `iff x in B^@`.
  2. 设 `{U_alpha: alpha in I}` 是 `A` 中的所有开集构成的子集族, 则 `x in A^@` `iff EE U in tau, x in U sube A` `iff EE alpha in I, x in U_alpha` `iff x in uuu_(alpha in I) U_alpha`.
  3. `A in tau iff A = uuu_(alpha in I) U_alpha = A^@`.
  4. `(A nn B) sube A rArr (A nn B)^@ sube A^@`, 同理 `(A nn B)^@ sube B^@`, 得到 `(A nn B)^@ sube A^@ nn B^@`. 另一方面, 由 3. 有 `A nn B supe A^@ nn B^@` `rArr (A nn B)^@ supe (A^@ nn B^@)^@` `= A^@ nn B^@`.
  5. 因为 `A^@ uu B^@` 是包含在 `A uu B` 中的开集, 由 2. 有 `(A uu B)^@ supe A^@ uu B^@`.
