计算机的基本组成

von Neumann 计算机的特点

  • 由运算器, 控制器, 存储器, 输入设备, 输出设备组成;
  • 指令和数据以同等地位存放于存储器内, 并按址寻访;
  • 指令和数据均以二进制数表示;
  • 指令由操作码和地址码组成, 操作码表示操作性质, 地址码指出操作数在内存的位置;
  • 指令在内存中顺序存放, 顺序执行. 特定条件下, 可以改变执行顺序;
  • 机器以运算器为中心, 输入输出设备与内存的数据传递通过运算器完成 (已弃用).

现代计算机三大部分

  • CPU, Central Processing Unit 中央处理器, 由运算器和控制器组成
  • MM, Main Memory 主存储器, 内存
  • I/O Equipment 输入输出设备
