网际协议——IP

    将网络互连起来的设备称为中间设备。中间设备按层次可以分为：
  • 物理层的转发器 (repeater)；
  • 数据链路层的网桥或桥接器 (bridge)；
  • 网络层的路由器 (router)；
  • 网络层以上的网关。

由于历史原因，许多文献把网络层的路由器称为网关。

分类的 IP 地址

类型 用途 开头 网络号 主机号
位数 可用 范围 位数 可用

D 类 多播 1110

计算可用网络数时，先减去固定开头位数，再减去保留的网络号。 计算可用主机数时，只需减去全 0 和 全 1 的主机号。

特殊的 IP 地址

  • 全 0: 本网络的本主机 (this);
  • 全 1: 在本网络上进行广播;
  • 主机号全 0: 网络 x 的地址;
  • 主机号全 1: 网络 x 上的所有主机;
  • 网络号全 0: 保留的 A 类地址，表示本网络的主机 x;
  • 127.x.x.x: 保留的 A 类地址, 用于本地软件环回测试;
  • 128.0.x.x: 保留的 B 类地址;
  • 192.0.0.x: 保留的 C 类地址.
