变量分离方程

可化为变量分离方程的类型

齐次方程

可化为齐次方程的

`dy/dx = (a_1 x + b_1 y + c_1)/(a_2 x + b_2 y + c_2)`, `|(a_1, b_1),(a_2, b_2)| != 0`

令 `alpha, beta` 是 `{(a_1 alpha + b_1 beta + c_1 = 0), (a_2 alpha + b_2 beta + c_2 = 0):}` 的解, 再作代换 `{(X = x - alpha),(Y = y - beta):}`, 即化为齐次方程.

线性方程

`dy/dx = p(x)y + q(x)`

先用分离变量法解齐方程 `dy/dx = p(x) y`, 得通解 `y = ce^(int p(x) dx)`, 再利用常数变易法, 设 `c = c(x)` 是 `x` 的函数, 将通解代入原方程比较, 得 `c'(x) = q(x) e^(-int p(x) dx)`, 积分得到 `c(x)` 的表达式.

Bernoulli 方程

`dy/dx = p(x)y + q(x)y^n`, `n >= 2`

显然 `y = 0` 是一个特解. `y != 0` 时, 两边同除以 `y^n`, 得 `y^(-n) dy/dx = p(x) y^(1-n) + q(x)`. 再令 `z = y^(1-n)` 化为线性方程.

恰当微分方程

`M(x, y) dx + N(x, y) dy = 0`, `(del M)/(del y) = (del N)/(del x)`

称 `mu(x, y)` 是 `M dx + N dy = 0` 的积分因子, 如果 `(del(mu M))/(del y) = (del(mu N))/(del x)`. 若 `mu` 只与 `x` 有关, 即 `(del mu)/(del y) = 0`, 则 `N (d mu)/dx = ( (del M)/(del y) - (del N)/(del x) ) mu`, 可以分离变量. 注意验证 `mu = 0` 是不是原方程的解.

一些常见的全微分 `d(xy) = ydx + xdy`,
`d(x/y) = (ydx - xdy)/y^2`,
`d(x^a y^b) = a x^(a-1) y^b dx + b y^(b-1) x^a dy`.
