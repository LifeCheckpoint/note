常数变易法

设方程 `x^((n)) + a_1(t)x^((n-1)) + cdots + a_(n-1)x' + a_n(t)x = f(t)` 或 `sum_(i=0)^n a_i(t) x^((n-i))(t) = f(t)`, `quad a_0(t) = 1` 对应的齐方程的基本解组为 `x_i(t)`, `quad i = 1, 2, cdots, n`. 我们解线性方程组 `[ (x_1(t), x_2(t), cdots, x_n(t)), (x_1'(t), x_2'(t), cdots, x_n'(t)), (vdots, vdots, , vdots), (x_1^((n-1))(t), x_2^((n-1))(t), cdots, x_n^((n-1))(t)) ][ (c_1'(t)), (c_2'(t)), (vdots), (c_n'(t)) ] = [ (0), (0), (vdots), (f(t)) ]`, 再积分, 就得到非齐方程的通解 `x = sum_(i=1)^n c_i(t) x_i(t)`.

解如下的常微分方程初值问题: `{ ( x'' + a^2 x = f(t) ), ( x|_(t=0) = x_0 ), ( x'|_(t=0) = x'_0 ) :}`

`{cos at, sin at}` 构成齐方程的基本解组. 设 `x = c_1 cos at + c_2 sin at`, 解线性方程组 `{ ( c_1' cos at + c_2' sin at = 0 ), ( c_1' sin at - c_2' cos at = -f/a ) :}` 得 `(c_1', c_2') = f/a (-sin at, cos at)`. 由初值条件有 `c_2|_(t=0) = (x_0')/a`, `quad c_1|_(t=0) = x_0`, 故 `c_1 = -1/a int_0^t f(tau) sin a tau "d"tau + x_0`,
`c_2 = 1/a (int_0^t f(tau) cos a tau "d"tau + x_0')`. 所以解为 `x = x_0 cos at + (x_0')/a sin at + 1/a int_0^t f(tau) sin[a(t-tau)] "d"tau`.

常系数齐次线性方程

`x^((n)) + a_1 x^((n-1)) + cdots + a_(n-1) x' + a_n x = 0`, `quad a_i in RR`, `i = 1, 2, cdots, n`.

设特征多项式 `lambda^n + a_1 lambda^(n-1) + cdots + a_(n-1) lambda + a_n` 的根 (即特征根) 为 `lambda_i`, 重数分别为 `n_i`, `i = 1, 2, cdots, l`, 且 `sum_(i=1)^l n_i = n`. 则微分方程的基本解组为 `t^j "e"^(lambda_i t)`, `quad j = 0, 1, cdots, n_i-1`, `quad i = 1, 2, cdots, l`. 若 `lambda_i = alpha + "i" beta` 为复根, 则 `bar(lambda_i) = alpha - "i" beta` 也必为特征根, 可以将基本解组中的一对共轭复根 `"e"^((alpha + "i" beta) t)` 与 `"e"^((alpha - "i" beta) t)` 换成 `"e"^(alpha t) cos beta t` 与 `"e"^(alpha t) sin beta t`.

Euler 方程

`t^n x^((n)) + a_1 t^(n-1) x^((n-1)) + cdots + a_(n-1) t x' + a_n x = 0`

将 `x = t^lambda` 代入方程, 得到 `n` 个特征根. 基本解组为 `ln^j |t| t^(lambda_i)`. `i, j` 的迭代范围与常系数齐次方程相同.

常系数非齐次线性方程

`x^((n)) + a_1 x^((n-1)) + cdots + a_(n-1) x' + a_n x = f(t)`, `quad a_i in RR`, `i = 1, 2, cdots, n`.

比较系数法

`f(t) = P_m(t) "e"^(lambda t)`, `quad P_m` 为 `m` 次多项式.

此时有特解 `Q_m(t) t^k "e"^(lambda t)`, `k` 为 `lambda` 在特征多项式中的重数, `Q_m(t)` 是系数待定的 `m` 次多项式.

`f(t) = A_m(t) "e"^(alpha t) cos beta t` 或 `f(t) = A_m(t) "e"^(alpha t) sin beta t`, `quad A_m` 为 `m` 次多项式.

此时有特解 `(P_m(t) cos beta t + Q_m(t) sin beta t) t^k "e"^(alpha t)`, `k` 为 `lambda = alpha + "i" beta` 在特征多项式中的重数 (注意到 `lambda` 及其共轭在特征多项式中的重数相等), `P_m(t), Q_m(t)` 为系数待定的 `m` 次多项式.

利用以上两种类型的非齐次项, 可以结合线性叠加原理, 写出更多非齐次项对应的特解.

解 Lanchester 方程 `{ (("d"A)/dt = -beta^2 B), (("d"B)/dt = -alpha^2 A) :}` 其中 `A(0) = A_0`, `B(0) = B_0`.

令 `f' = B`, 且 `f(0) = 0`. 原方程组化为 `{ (("d"A)/dt = -beta^2 ("d"f)/dt), (("d"^2 f)/dt^2 = -alpha^2 A) :}` 第一式两边积分得 `A = -beta^2 f + A_0`, 代入第二式就有 `f'' = alpha^2 beta^2 f - alpha^2 A_0`, 它的解设为 `f = c_1 "e"^(alpha beta t) + c_2 "e"^(-alpha beta t) + c_3`. 代入前式得 `c_3 = A_0/beta^2`. 又 `B = f' = alpha beta (c_1 "e"^(alpha beta t) - c_2 "e"^(-alpha beta t))`, 由 `f(0) = 0` 和 `B(0) = B_0` 有 `{ (c_1 + c_2 = -A_0/beta^2), (c_1 - c_2 = B_0/(alpha beta)) :}` 解得 `c_(1, 2) = 1/(2 beta) (-A_0/beta +- B_0/alpha)`. 最终 `A = A_0 cosh(alpha beta t) - B_0 beta/alpha sinh(alpha beta t)`,
`B = -A_0 alpha/beta sinh(alpha beta t) + B_0 cosh(alpha beta t)`.

从相平面上看, 有 `("d"A)/("d"B) = (beta^2 B)/(alpha^2 A)`, 解为 `alpha^2 A^2 = beta^2 B^2 + c`, 是双曲线.

Laplace 变换

`cc L[f(t)] = F(s) = int_0^oo "e"^(-st) f(t) dt`, `quad "Re"(s) gt sigma`.

若 `cc L[x(t)] = X(s)`, 则 `cc L[x'(t)]` 存在且等于 `sX(s) - x(0)`. 由归纳法知 ` cc L[x^((n))(t)]` `= s^n X(s) - sum_(k=0)^(n-1) x^((k))(0) s^(n-1-k)` `= s^n X(s) - sum_(k=0)^(n-1) x^((n-1-k))(0) s^k`. Laplace 变换将方程 `sum_(i=0)^n a_i x^((n-i))(t) = f(t)` 变为 ` X(s) sum_(i=0)^n a_i s^(n-i)` `= F(s) + sum_(i=0)^(n-1) a_i sum_(k=0)^(n-i-1) x^((k))(0) s^(n-i-1-k)` `= F(s) + sum_(k=0)^(n-1) x^((k))(0) sum_(i=0)^(n-1-k) a_i s^(n-1-k-i)` `= F(s) + sum_(k=0)^(n-1) x^((n-1-k))(0) sum_(i=0)^k a_i s^(k-i)`. 记 `p_k(x) = sum_(i=0)^k a_i s^(k-i)`, 上式可以写成 `X(s) p_n(s) = F(s) + sum_(k=0)^(n-1) x^((n-1-k))(0) p_k(s)`. 解出 `X(s)` 后, 适当分解分式, 再作 Laplace 反变换 `f(t) = 1/(2 pi i) lim_(T to oo) int_(gamma-iT)^(gamma+iT) "e"^(st) F(s) "d"s`, 其中 `gamma in RR` 使得积分路径位于 `F(s)` 的收敛域内.

`cc L[t^p "e"^(zt)] = (Gamma(p+1))/(s-z)^(p+1)`, `quad p in (-1, +oo)`, `z in CC`, `"Re"(s) gt "Re"(z)`. 从而 `cc L["e"^(zt)] = 1/(s-z)`. 上式取 `z = a "i"`, 有 `cc L[cos at + "i" sin at]` `= 1/(s-a"i")` `= (s+a"i")/(s^2 + a^2)`. 分别取实部与虚部, 即 `cc L[cos at] = s/(s^2+a^2)`, `quad` `cc L[sin at] = a/(s^2+a^2)`.

用 Laplace 变换解.

对方程两边作 Laplace 变换得 `X(s) (s^2+a^2) = x_0 s + x_0' + cc L[f(t)]`,
`X(s) = (x_0 s)/(s^2+a^2) + (x_0')/(s^2+a^2) + (cc L[f(t)])/(s^2+a^2)`. 作 Laplace 反变换得 `x = x_0 cos at + (x_0')/a sin at + int_0^t f(tau) sin [a(t-tau)]"d"tau`.

降阶法

n 阶方程 `F(t, x^((k)), x^((k+1)), cdots, x^((n))) = 0`

此方程不显含 `k` 阶以下的导数. 令 `y = x^((k))`, 方程降低 k 阶.

n 阶方程 `F(x, x', cdots, x^((n))) = 0`

此方程不显含自变元, 令 `y = x'`, 降低 1 阶.

齐次线性微分方程 `sum_(i=0)^n a_i(t) x^((n-i))(t) = 0`

若有非零特解 `x_1, cdots, x_k`, 则令 `x = x_k int z dt`, 化为 `z` 的 `n-1` 阶方程, 有解 `z_i = ((x_i)/(x_k))^'`, `i = 1, 2, cdots, k-1`.

幂级数解法

将 `y = sum_(n=0)^oo a_n x^n`,
`y' = sum_(n=1)^oo n a_n x^(n-1) = sum_(n=0)^oo (n+1) a_(n+1) x^n`,
`y^((k)) = sum_(n=k)^oo n(n-1)cdots(n-k+1) a_n x^(n-k)` `= sum_(n=0)^oo (n+k)cdots(n+2)(n+1) a_(n+k) x^n` 代入方程并比较系数, 注意幂级数的收敛半径.
