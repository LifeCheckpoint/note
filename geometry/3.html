线段的定比分点

    设 `AB` 为一线段 (`A != B`). 如果点 `C` 满足 `vec(AC) = lambda vec(CB)`, 则称点 `C` 分线段 `AB` 成定比 `lambda`.
  1. 若 `lambda = -1`, 得到 `vec(AB) = 0`, 矛盾, 故 `lambda != -1`.
  2. 若 `lambda gt 0`, `vec(AC)`, `vec(CB)` 同向, 称 `C` 为内分点;
  3. 若 `lambda lt 0` 且 `lambda != -1`, `vec(AC)`, `vec(CB)` 反向, 称 `C` 为外分点;
  4. 若 `lambda = 0`, `C = A`;
  5. `lambda to +oo` 时, `C to B`;

设点 `C` 分 `A(a_1, a_2, a_3)`, `B(b_1, b_2, b_3)` 成定比 `lambda != -1`. 则 `vec(OC) = 1/(1+lambda) (vec(OA) + lambda vec(OB))`. 特别取 `lambda = 1`, 知 `AB` 中点 `M` 满足 `vec(OM) = 1/2 (vec(OA) + vec(OB))`.

    设 `P, Q, R` 分别分 `triangle ABC` 的边 `AB, BC, CA` 成定比 `lambda, mu, nu`, 则
  1. Menelaus 定理 `P, Q, R` 共线当且仅当 `lambda mu nu = -1`;
  2. Ceva 定理 若 `P, Q, R` 都是内分点, 则 `AQ, BR, CP` 共点当且仅当 `lambda mu nu = 1`.
  2. Ceva 定理的另一证法是利用等式 ` lambda mu nu = (AP * BQ * CR)/(BP * CQ * AR) = (S_(AMC) * S_(BMA) * S_(CMB)) / (S_(BMC) * S_(CMA) * S_(AMB)) = 1`.

三角形的重心

由 Ceva 定理知, 三角形三条中线交于一点, 称这点为三角形的重心.

    (图??) 设 `G` 是 `triangle ABC` 的重心, `P, Q, R` 分别是边 `AB, BC, CA` 的中点. 则
  1. 三条中线将三角形分成了面积相等的 6 个小三角形;
  2. `G` 分三条中线 `AQ, BR, CP` 成定比 2, 从而 `vec(AG) = 1/3 (vec(AB) + vec(AC))`, `vec(BG) = 1/3 (vec(BC) + vec(BA))`, `vec(CG) = 1/3 (vec(CA) + vec(CB))`.
  3. `vec(AG) + vec(BG) + vec(CG) = bb 0`.
  4. `vec(OG) = 1/3 (vec(OA) + vec(OB) + vec(OC))`, `O` 是平面上任一点.
  1. 略.
  2. 由 1, `S_(ABG) = 2 S_(QBG)`, 所以 `AG = 2GQ`; 因而 ` vec(AG) = 2/3 vec(AQ) = 2/3 * 1/2 (vec(AB) + vec(AC))` `= 1/3 (vec(AB) + vec(AC))`. 其余各式同理.
  3. 将 2 的三式相加即得证.
  4. 直接计算 `{: ( vec(OG) ,= vec(OA) + vec(AG) ),( ,= vec(OA) + 1/3 (vec(AB) + vec(AC)) ),( ,= vec(OA) + 1/3 (vec(OB) - vec(OA) + vec(OC) - vec(OA)) ),( ,= 1/3 (vec(OA) + vec(OB) + vec(OC)). ):}`

三角形的内心

    (图??) 设在 `triangle ABC` 中, `CP` 平分 `angle ACB`, `a, b` 分别是 边 `BC, CA` 的长. 则
  1. `(AP)/(BP) = (sin angle B)/(sin angle A) = b/a`;
  2. `vec(CP) = 1/(a+b) (a vec(CA) + b vec(CB))`.
  1. 由正弦定理, ` (AP)/(sin angle ACP) = (CP)/(sin angle A) = (AC)/(sin angle APC)`,
    ` (BP)/(sin angle BCP) = (CP)/(sin angle B) = (BC)/(sin angle BPC)`.     注意 `angle ACP = angle BCP` 和 `angle APC + angle BPC = pi` 即得证.
  2. 直接计算, ` vec(CP) = vec(CA) + b/(a+b) vec(AB)` `= vec(CA) + b/(a+b) (vec(CB) - vec(CA))` `= 1/(a+b) (a vec(CA) + b vec(CB))`.

由 Ceva 定理和知, 三角形的三条角平分线交于一点, 称为三角形的内心.

    另一个证明三条角平分线共点的思路是利用如下命题: 设 `l` 是过角的顶点的直线, 则以下三款等价:
  1. `l` 平分角;
  2. `l` 上任意一点到角两边的距离相等;
  3. `l` 上存在一点到角两边的距离相等.
    4. 容易证明, 若三角形的两条角平分线交于一点, 则这点与第三点的连线是第三条角平分线.
    设 `I` 是 `triangle ABC` 的内心, `P, Q, R` 分别是三条角平分线与 `AB, BC, CA` 的交点. 则
  1. 三条角平分线将 `triangle ABC` 分成的 6 个小三角形的面积与其在 `triangle ABC` 边上的那一条边长成正比.
  2. `I` 分别分三条角平分线 `AQ, BR, CP` 成定比 `(b+c)/a`, `(c+a)/b`, `(a+b)/c`, 从而 `vec(AI) = 1/(a+b+c) (b vec(AB) + c vec(AC))`,
    `vec(BI) = 1/(a+b+c) (c vec(BC) + a vec(BA))`,
    `vec(CI) = 1/(a+b+c) (a vec(CA) + b vec(CB))`.
  3. `vec(OI) = 1/(a+b+c) (a vec(OA) + b vec(OB) + c vec(OC))`, `O` 是平面上任一点.
  1. 略.
  2. 在 `triangle ACP` 中应用 有 ` (CI)/(IP) = (AC)/(AP) = b/(c * b/(a+b)) = (a+b)/c`. 因而 ` vec(CI) = (a+b)/(a+b+c) vec(CP)` `= 1/(a+b+c) (a vec(CA) + b vec(CB))`. 其余各式同理.
  3. 直接计算 `{: ( vec(OI) ,= vec(OC) + vec(CI) ),( ,= vec(OC) + 1/(a+b+c)(a vec(CA) + b vec(CB)) ),( ,= vec(OC) + 1/(a+b+c)[a (vec(OA)-vec(OC)) + b (vec(OB)-vec(OC))] ),( ,= 1/(a+b+c) (a vec(OA) + b vec(OB) + c vec(OC)). ):}`

三角形的垂心

三角形的三条高线 (高所在的直线) 交于一点, 称为垂心.

设 `triangle ABC` 的两条高线 `AQ, BR` 交于点 `H`, 下证 `CH _|_ AB`. 由 `AH _|_ BC`, `BH _|_ CA` 有 `0 = vec(AH) * vec(BC) = vec(AH)*vec(HC) - vec(AH)*vec(HB)`,
`0 = vec(BH) * vec(CA) = vec(BH)*vec(HA) - vec(BH)*vec(HC)`. 两式相加, `0 = vec(AH)*vec(HC) - vec(BH)*vec(HC) = vec(AB)*vec(HC)`, 即 `CH _|_ AB`.
