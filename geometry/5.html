基本概念

平面上到定直线和到定点距离相等的动点轨迹称为抛物线 (parabola). 这条定直线称为准线, 这个定点称为焦点. 焦点到准线的垂线段称为焦准距.

图5-1

下文中如无特别说明 "在空间中", 我们的问题总是限于一个平面内的.

曲线上两点的连线称为弦. 称一条曲线关于一直线对称, 如果对曲线上的任一点, 存在这曲线上的一点, 使这两点位于直线异侧, 且连结它们的弦被这直线垂直平分. 这条直线称为曲线的对称轴, 曲线与轴的交点叫做顶点.

已知焦点与准线, 作出抛物线上的点.

图5-2

设 `S` 是焦点, 作直线 `SX` 垂直准线于 `X`. 取 `SX` 中点 `A`, 则 `AX = AS`. 由定义, `A` 在抛物线上.
在射线 `AS` 上任取一点 `N`, 作直线 `PNP'` 垂直 `SX` 于 `N`. 以 `S` 为心, `XN` 为半径作圆, 与 `PNP'` 相交于 `P` 和 `P'`. 作 `PM`, `P'M'` 分别垂直准线于 `M`, `M'`. 则由于 `PS = NX = PM`, 所以 `P` 在抛物线上. 同理 `P'` 也在抛物线上. `XS` 垂直于圆 `S` 的弦, `PP'`, 故 `PP'` 被直线 `XS` 垂直平分, 即 `P` 与 `P'` 关于直线 `XS` 对称.
于是射线 `AS` 上的每一点都对应着抛物线上的一组对称点. 另外容易说明, 若 `N` 与 `S` 在 `A` 的异侧, 则作出的圆与直线 `PNP'` 无交点. 这表示抛物线完全落在顶点的一侧.

过焦点且垂直于准线的直线是抛物线的唯一对称轴.

抛物线上, 一点到焦点的连线叫做焦半径, 通过焦点的弦叫做焦点弦. 抛物线上一点到轴的垂线段叫做这个点的纵标线, 这一点与它的对称点的连线叫双纵标线; 特别, 通过焦点的双纵标线称为正焦弦. 轴在顶点和纵标线之间的部分叫做横标线.

若 `S` 是抛物线的焦点, `A` 是抛物线的顶点, 纵标线 `PN` 与抛物线交于 `N`, 则 `PN^2 = 4 AS * AN`; 特别 `N = S` 时, 得到正焦弦的一半 `PN = 2 AS`.

图5-3

设抛物线的轴与准线的交点为 `X`. 连接 `PS`, 则 `PS = NX = AX + AN = AS + AN`, 且不论 `N` 在线段 `AS` 上, 还是在 `AS` 延长线上, 都有 `NS^2 = (AS - AN)^2`. 所以 ` PN^2 = PS^2 - NS^2 = (AS + AN)^2 - (AS - AN)^2` `= 4 AS * AN`.

抛物线的切线

设 `PP'` 是曲线上的弦. 若存在过 `P` 的直线 `l`, 使得 `P'` 沿曲线趋于 `P` 时, 直线 `PP'` 与 `l` 的夹角趋于 0, 则称 `l` 为曲线在 `P` 点处的切线.

曲线上一点的切线若存在, 则必唯一.

(图5-4) 若 `S` 是抛物线的焦点, `PP'` 是焦点弦, `Q` 是抛物线上异于 `P, P'` 的一点, 弦 `PQ` 延长后交准线于 `K`. 则 `SK` 平分 `triangle PQS` 的外角 `angle P'SQ`.

图5-4
图5-5

分别作 `P, Q` 到准线的垂线, 垂足为 `M, N`. 由三角形相似与抛物线定义得 `(KQ)/(KP) = (QN)/(PM) = (SQ)/(SP)`. 过 `P` 作直线 `SK` 的平行线, 与直线 `SQ` 交于 `R`, 则 `triangle PQR ~ triangle KQS`, 有 `(KQ)/(KP) = (SQ)/(SR)`. 所以 `SP = SR`. 从而 ` angle P'SK = angle SPR = angle SRP = angle QSK`.

    (图5-5) 设 `S` 是抛物线的焦点; `P` 是抛物线上异于顶点的一点, `PM` 垂直准线于 `M`; `K` 是准线上异于 `M` 的一点, 则下面各款等价:
  1. `PK` 是抛物线的切线;
  2. `angle PSK` 是直角;
  3. `triangle SPK ~= triangle MPK`;
  4. `PK` 平分 `angle SPM`.

在引理5-1中令 `Q` 沿曲线趋于 `P`, 则 `angle PSQ` 趋于 0, `angle P'SQ` 趋于平角, `angle PSK` 就趋于直角. 由此 1, 2 等价.
由三角形全等的定理 (斜边, 直角边) 与 (两边一夹角) 容易证明余下的部分.

抛物线顶点处的切线与准线平行.

抛物线的切线与抛物线有唯一的公共点.

Adams 性质 设 `S` 是抛物线的焦点, `P` 是抛物线上一点, `O` 在过 `P` 点的切线上. 作 `OI` 垂直准线 `I`, `OU` 垂直焦半径 `SP` 于 `U`, 则 `OI = SU`.

图5-6

作 `PM` 垂直准线于 `M`; 设 `PO` 交准线于 `K`, 连接 `SK`. 由定理5-2, `OU` 平行于 `KS`. 由相似关系易得 `(OI)/(PM) = (OK)/(PK) = (SU)/(SP)`. 但 `PM = SP`, 于是 `OI = SU`.

已知焦点和准线, 从抛物线外一点 `O` 作抛物线的两条切线.

图5-7

设 `S` 是抛物线的焦点. 以 `O` 为心, `OS` 为半径作圆, 若 `OS` 不小于焦准距的一半, 此圆将交准线于 `M, N` 两点. 过 `M, N` 作准线的垂线, 分别与抛物线交于 `P, Q`. 则 `OP`, `OQ` 为所求的切线.
利用三角形全等的定理 (三边相等) 容易证明 `triangle OMP ~= triangle OSP`, 从而 `PO` 平分 `angle SPM`, 由定理5-2, `PO` 是 `P` 点处的切线. 同理可证 `QO` 是 `Q` 点处的切线.

抛物线焦点弦两端的切线相互垂直, 且交点在准线上.

图5-8

设 `S` 是抛物线的焦点, `PQ` 是焦点弦; 直线 `PK` 切抛物线于点 `P`, 交准线于 `K`.
由定理5-2, `PQ _|_ SK`. 再由定理5-2, `QK` 是抛物线的切线. 这说明两切线的交点在准线上. 下证它们相互垂直. 分别作 `PM, QN` 垂直准线于 `M,N`, 由定理5-2, `triangle SPK ~= triangle MPK`, 于是 `angle PKS = 1/2 angle MKS`. 同理 `angle QKS = 1/2 angle NKS`, 相加得 `angle PKQ = 1/2 angle MKN = pi/2`.

设过曲线上一点的切线存在, 过这一点, 且与切线垂直的直线称为曲线在该点的法线. 法线如果存在, 也必定唯一.

抛物线上一点的切线与纵标线在轴上所截的线段称为抛物线的次切线, 抛物线上一点的法线与纵标线在轴上所截的线段称为抛物线的次法线.

设 `S` 是抛物线的焦点, `P` 是抛物线上异于顶点的一点. 切线 `PT` 与轴交于 `T`, 法线 `PG` 与轴交于 `G`, 则 `ST = SP = SG`.

图5-9

作 `PM` 垂直准线于 `M`, 则由定理5-2, `angle STP = angle MPT = angle SPT`. 所以 `ST = SP`; 再由 `PT _|_ PG` 容易证明 `angle SPG = angle SGP`, 所以 `SP = SG`.

  1. 抛物线的次切线被顶点平分, 换言之, 次切线是横标线的两倍;
  2. 抛物线的次法线等于焦准距.
    (图5-9) 设 `S` 是抛物线的焦点, `A` 是抛物线的顶点; `PT` 切抛物线于 `P`, 交轴于 `T`.
  1. 由定理5-5, `ST = SP = XN`. 而 `AS = AX`, 两边相减得 `AT = AN`.
  2. 由定理5-5, `SG = SP = XN`. 两边同减 `SN` 得 `NG = XS`.
    定理5-5 中, 若抛物线的顶点 `A` 处的切线与 `PT` 交于 `Y`, 则
  1. `SY` 垂直平分 `PT`;
  2. `SY` 是 `SA` 与 `SP` 的比例中项, 即 `SY^2 = SA * SP`.
  1. 作 `P` 的纵标线 `PN`, 由推论5-6, `A` 平分次切线 `TN`. 由推论5-2, `AY` 与 `PN` 平行, 于是 `Y` 平分 `PT`. 由定理5-5, `STP` 是等腰三角形, 故 `SY` 垂直平分 `PT`.
  2. 利用 `triangle SAY ~ triangle SYP` 即可.
