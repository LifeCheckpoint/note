可解群

单群

设 `G` 为非平凡群, 称 `G` 为单群, 如果它 只有平凡的正规子群 `{e}` 和 `G`, 即 `H normal G rArr H = {e} or H = G`. 其中 `e` 为 `G` 的幺元.

Abel 群 `G` 为一单群当且仅当它为一素数阶循环群.

因为 Abel 群的每个子群都是正规子群, 所以 `G` 为一单群当且仅当它没有非平凡的子群, 从而当且仅当 `G` 为一素数阶循环群. 事实上, 由 Lagrange 定理, 素数阶循环群显然无非平凡子群. 另一方面, 由 `G` 无非平凡子群知 `G` 中非幺元都是 `|G|` 阶的, 故 `G` 为循环群 `(:a:)`. 若 `|G| = mn`, `m, n gt 1`, 则 `(:a^m:)` 为一非平凡子群, 故 `|G|` 为素数.

单群在群同态映射下的像也为一单群.

不妨设 `f: G to G'` 为一群满同态 (否则 `f` 是到 `f(G)` 上的满同态). 取 `G'` 的正规子群 `N' != {e'}`, 则 `f^-1(N') normal G`, 但 `G` 为单群, 故 `f^-1(N')` 等于 `{e}` 或 `G`. 但由 `f` 是满射知道, `N' = f(f^-1(N')) != {e'}`, 故只能 `f^-1(N') = G`. 再由 `f` 是满射, `N' = f(f^-1(N')) = f(G) = G'`. 所以 `G'` 为单群.

  2. 若 `pi` 的分解式中只有对换, 则 `pi^-1 = pi`. 由于 `pi` 至少使 `3` 个点发生了移动, 不妨令 `pi = (12)(34) cdots`. 取 `sigma = (345) in A_n`. 显然除了 `5`, `pi` 的不动点也是 `tau := pi^-1 sigma pi sigma^-1 = pi (12)(45) cdots in H` 的不动点. 显然 `tau != (1)`, 否则推出 `pi = pi^-1 = sigma`. 又 `tau(1) = 1`, `tau(2) = 2`, 而 `pi(1) = 2`, `pi(2) = 1`, 因此 `f(tau) gt f(pi)`, 一个矛盾.

子群列

令 `G` 为非平凡群, `M ⊲ G` (即 `M normal G` 且 `M != G`). 称 `M` 为 `G` 的一个极大正规子群, 如果包含 `M` 的 `G` 的正规子群只有 `M` 和 `G`, 即 `(AA N normal G)` `M sub N rArr N = G`.

任意非平凡有限群 `G` 都有极大正规子群.

`|G| = 2` 时, `{e}` 即是 `G` 的极大正规子群; `|G| = n gt 2` 时, 任取 `M_1 ⊲ G`, 若 `M_1` 不极大, 则存在 `M_2 ⊲ G`, `M_1 sub M_2`. 这蕴含 `|M_1| lt |M_2|`. 若 `M_2` 不极大, 类似可得 `M_3, M_4, cdots`, 且 `|M_1| lt |M_2| lt |M_3| lt cdots`. 但 `|G|` 有限, 这一序列也有限, 所以存在 `G` 的极大正规子群.

若群 `M ⊲ G`, 则 `M` 为 `G` 的一个极大正规子群当且仅当 `G//M` 为一单群.

充分性: 任取 `N normal G`, 且 `M sub N`, 由 `M ⊲ G` 知 `M ⊲ N`. 考察 `N//M = {n M | n in N} le G//M`. 因为 `N normal G`, 所以 `N//M normal G//M`, 但 `G//M` 为单群, 又由 `M sub N`, `N//M != {M}`, 故 `N//M = G//M`, 即 `N = G`.
必要性: 作自然同态 `eta: g in G to g M in G//M`. 取 `H normal G//M`, 且 `H != {M}`, 又记 `K = eta^-1(H) normal G`. 由 `{M} sub H`, `M = eta^-1({M}) sub eta^-1(H) = K`. 但 `M` 是极大正规子群, 故 `K = G`, `H = eta(K) = G//M`.

令 `G` 为非平凡群, `k ge 1`. 称 `G` 的子群列 `G = G_0 ⊳ G_1 ⊳ cdots ⊳ G_k = {e}` 为 `G` 的一个长度为 `k` 的合成群列, 如果各个合成因子 `G_i//G_(i+1)` 为一单群, `i = 0, 1, cdots, k-1`.

`G_i//G_(i+1)` 为单群, 则 `G_(i+1)` 为 `G_i` 的一个极大正规子群. 这保证了合成群列中不能再插入其他子群.

任意非平凡有限群 `G` 有合成群列.

`|G| = 2` 时, `G ⊳ {e}` 即为 `G` 的一个合成群列. `|G| = n gt 2` 时, 假设阶小于 `n` 的非平凡有限群有合成群列, 考查阶为 `n` 的情形. 若 `G` 为单群, 则 `G ⊳ {e}` 为 `G` 的一个合成群列; 否则取 `G` 的极大正规子群 `M`, 显然 `1 lt |M| lt |G|`. 由归纳假设, `M` 有合成群列. 又 `G//M` 为单群, 可以将 `G` 加入到 `M` 的合成群列中, 成为 `G` 的一个合成群列.

令 `G` 为一群, `k ge 1`, 称 `G` 的子群列 `G = G_0 ⊵ G_1 ⊵ cdots ⊵ G_k = {e}` 为 `G` 的一个长度为 `k` 的次正规群列, `G_i//G_(i+1)` 称为 `G` 的次正规因子; 若进一步 `G_i normal G`, 则上述群列称为 `G` 的正规群列, `G_i//G_(i+1)` 称为 `G` 的正规因子; `i = 0, 1, cdots, k-1`.

合成群列是次正规群列, 正规群列是次正规群列.

自同构群

令 `G` 为一群, `G` 上的全体自同构映射 `"Aut"(G)` 关于变换的通常合成为一群, 称为 `G` 的自同构群.

    令 `G = (:a:)` 为一循环群, 则
  1. `"Aut"(G)` 为 Abel 群;
  2. 若 `|G| = n`, 则 `"Aut"(G)` 为一 `varphi(n)` 阶 Abel 群, `varphi(n)` 为 Euler 函数;
  3. 若 `|G| = oo`, 则 `"Aut"(G)` 为一 2 阶 Abel 群 `{i, sigma}`, 其中 `i(a) = a`, `sigma(a) = a^-1`.
  1. `AA sigma, tau in "Aut"(G)`, 其中 `sigma(a) = a^m`, `tau(a) = a^n`, `m, n in ZZ`, 则 `AA a^k in G`, `sigma tau(a^k) = a^(m n k) = tau sigma(a^k)`.
  2. 由于每一对生成元之间的对应关系都决定了 `G` 的一个自同构 (如 `sigma(a) = b`, 则 `sigma(a^k) = b^k`), 从而 `|"Aut"(G)|` 等于 `G` 中生成元的个数, 即 `varphi(n)`.
  3. 无限阶循环群有两个生成元 `a^(+-1)`, 所以 `|"Aut"(G)| = 2`.

令 `G` 为一群, `a in G`, 定义由 `a` 确定的 `G` 的内自同构如下: `tau_a: G to G`
`g to a g a^-1` 可以验证 `tau_a in "Aut"(G)`. `G` 的全体内自同构的集合记为 `"Inn"(G)`, 可以验证 `"Inn"(G) normal "Aut"(G)`, 称 `"Inn"(G)` 为 `G` 的内自同构群, `"Aut"(G)//"Inn"(G)` 为 `G` 的外自同构群.

  1. 显然 `tau_a` 为双射, 又 `tau_a(g h) = a(g h)a^-1 = a g a^-1 a h a^-1 = tau_a(g)tau_a(h)`, `tau_a` 为一同构, 故 `tau_a in "Aut"(G)`.
  2. 显然 `"Inn"(G) sube "Aut"(G)`, 又 `AA tau_a, tau_b in "Inn"(G)`, `tau_a^-1 = tau_(a^-1) in "Inn"(G)`, `quad tau_a tau_b = tau_(a b) in "Inn"(G)`. 所以 `"Inn"(G) le "Aut"(G)`.
  3. `AA tau_a in "Inn"(G), sigma in "Aut"(G), x in G`, `(sigma tau_a sigma^-1)(x)` `= sigma( tau_a (sigma^-1(x)))` `= sigma(a sigma^-1(x) a^-1)` `= sigma(a) x sigma(a)^-1` `= tau_(sigma(a))(x)`. 从而 `sigma tau_a sigma^-1 = tau_(sigma(a)) in "Inn"(G)`, 于是 `"Inn"(G) normal "Aut"(G)`.

令 `G` 为一群, 则 `"Inn"G ~= G//Z(G)`.

作 `eta: a in G to tau_a in "Inn"(G)`, 由内自同构的定义知 `eta` 为一满射, 又 `AA g in G`, `tau_(a b)(g) = (a b)g(a b)^-1 = (tau_a tau_b)(g)`, 故 `eta` 为一满同态.
下证 `"Ker"eta = Z(G)`. 这是因为 `a in "Ker"eta` `iff tau_a = 1_G` `iff (AA x in G) x = tau_a(x) = a x a^-1` `iff (AA x in G) x a = a x` `iff a in Z(G)`. 根据同态基本定理, `"Inn"(G) = "Im"eta ~= G//"Ker"eta = G//Z(G)`.

令群 `H le G`. 定义 `H` 的正规化子和中心化子如下: `N_G(H) := {g in G | g H = H g}`,
`Z_G(H) := {g in G | (AA h in G) g h = h g}`. 容易验证 `N_G(H), Z_G(H) le G`. 由定义可见, `N_G(H)` 是 `G` 中所有 `H` 可吸收的元素, `Z_G(H)` 是 `G` 中所有与 `H` 可换的元素.

    设群 `H le G`, 则
  1. `H normal N_G(H)`; 特别 `H normal G iff N_G(H) = G`;
  2. `Z_G(H) normal N_G(H)`; 特别 `H normal G` 时, `Z_G(H) normal G`;
  3. 若 `K normal H`, 则 `H le N_G(K)`.

`N_G(H) = H` 时, `H` 在 `G` 中的正规程度最差; `N_G(H) = G` 时, `H normal G`, 在 `G` 中的正规程度最好. `Z_G(H) = H` 时, `H` 为一 Abel 群; `Z_G(H) = G` 时, `H le Z(G)`. 另外显然 `Z(G) = Z_G(G)`.

令群 `H le G`, 则 `N_G(H) // Z_G(H) ~= "Inn"(N_G(H))|_H le "Aut"(H)`

令 `g in N_G(H)`, 则 `tau_g(H) = g H g^-1 = H`, 其中 `tau_g in "Inn"(N_G(H))`. 因此 `tau_g|_H` 为 `H` 上一自同构. 据此, 存在同态 `eta: N_G(H) to "Aut"(H)`
`g to tau_g|_H`. 又 `"Ker"eta = {g in N_G(H) | tau_g|_H = 1_H}` `= {g in N_G(H) | (AA h in H) g h g^-1 = h}` `= Z_G(H) nn N_G(H)` `= Z_G(H)`. 于是由群同态基本定理 `N_G(H)//Z_G(H)` `= N_G(H)//"Ker"eta ~= "Im"eta` `= Inn(N_G(H))|_H le` `"Aut"(H)`.

令群 `H normal G`, 则 `N_G(H) = G`, 于是 `G//Z_G(H) ~= "Inn"(G)|_H le "Aut"(H)`.

在上面的推论中取 `H = G`, 可以看到是的推广.

令 `G` 为一群, `a in G`, 且 `(:a:) normal G`. 则 `G' sube Z_G((:a:))`.

由, `G//Z_G((:a:)) ~= "Inn"(G)|_(:a:) le "Aut"((:a:))` 后者是循环群的自同构群, 必为 Abel 群, 所以 `G//Z_G((:a:))` 也是 Abel 群. 从而 `G' sube Z_G((:a:))`.

群作用

    令 `X` 为一非空集, `e` 为群 `G` 的幺元. 称映射 `f: G xx X to X`
    `(g, x) to f(g, x) := g(x)`     为群 `G` 在 `X` 上的作用, 如果 `f` 满足
  1. `(AA g, h in G and AA x in X)` `(g h)x = g(h x)`;
  2. `(AA x in X)` `e x = x`.
    3. 由两条性质立即推出 `g^-1(g(x)) = (g^-1 g)x = e x = x`.

令 `X` 为一非空集, `G` 为一群. 则 `G` 在 `X` 上的一个群作用 `f(g, x)` 确定 `G` 到 `X` 上对称群 `S_X` (`S` 上全体可逆变换) 的一个群同态 `eta`, 反之亦然.
