集合的概念

集合, 又叫做类, 族或搜集, 是数学中极少数不加定义的 "元概念" 之一, 而且是 "元概念" 里最基本, 最重要的一个. 一般把集合理解为具备某个性质的对象的全体.

    集合具有:
  • 确定性. 对于给定的集合 `A` 和对象 `a`, 一定有 `a` 属于 `A` 和 `a` 不属于 `A` 之一成立, 分别记为 `a in A`, `a !in A`. 当 `a in A` 时, 称 `a` 是 `A` 的元素或成员或点.
  • 互异性. 集合中没有两个相同的元素, 或者说, 任意元素在集合中都只出现一次. 在形式上, 有时我们会把一个集合写成 `{x, y, x}`, `{x, x, y, y}`, 等. 它们和 `{x, y}` 是相同的集合.
  • 无序性. 元素在集合中的出现次序是无关紧要的, 比如 `{1, 2, 3}` 和 `{2, 3, 1}` 是同一个集合.
    • 互异性和无序性是下面的外延公理的推论.

下面我们用大写字母 `A, B, C` 等表示集合, 小写字母 `a, b, c` 等表示集合的元素.

集合的表示

    由集合的确定性可知, 只要能够确定集合中的元素, 也就确定了一个集合. 一个集合可以有多种表示方式.
  1. 列举法. 如 `A = {a, b, c}`, `B = {1, 2, cdots, 10}`, `C = {{a}, {b}, {c}}`. 这里 `C` 是集合的集合, 与 `A` 不同. 同理单元素集合 `{a}` 与元素 `a` 不同, 应区分对待.
  2. 描述法. 用谓词描述集合元素的公共特征. `S = {x: P(x)}` 表示 `x in S` 当且仅当命题 `P(x)` 成立.

集合的基本关系

  1. 包含 `A sube B iff (a in A rArr a in B) iff (AA a) a in A rarr a in B`. 此时称 `A` 含于 `B`, 或 `B` 包含 `A`, 或 `A` 是 `B` 的子集;
  2. 相等 (外延公理) `A = B iff A sube B and B sube A`. 此时称 `A` 与 `B` 相等; 依上述定义, 两个集合相等当且仅当它们所含的元素相同, 而与元素的次序和重复次数无关, 如 `{a, b, c} = {a, c, b} = {a, b, c, b}`.
  3. 真包含 `A subne B iff A sub B iff A sube B and A != B`. 此时称 `A` 真包含于 `B`, 或 `B` 真包含 `A`, 或 `A` 是 `B` 的真子集.

不含任何元素的集合称为空集 `O/`. 在集合论的各种场合下, 往往有一个集合 `Omega` 包含了我们讨论的所有集合. 称 `Omega` 为全集或论述域.

若 `I` 是一集合, 且任意 `alpha in I` 唯一确定一个集合 `A_alpha`, 则称集合 ` {A_alpha: alpha in I} := {B: (EE alpha in I) B = A_alpha}` 为集合族.

基本运算

并/交 `A uu B := {x: x in A or x in B}`,
`A nn B := {x: x in A and x in B}`. `uuu_(alpha in I) A_alpha := {x: EE alpha in I, x in A_alpha}`,
`nnn_(alpha in I) A_alpha := {x: AA alpha in I, x in A_alpha}`.

  1. 蕴含式 `AA alpha in I, A_alpha sube B iff uuu_(alpha in I) A_alpha sube B`,
    `AA alpha in I, B sube A_alpha iff B sube nnn_(alpha in I) A_alpha`.
  2. 分配律 `A nn (uuu_(alpha in I)B_alpha) = uuu_(alpha in I)(A nn B_alpha)`,
    `A uu (nnn_(alpha in I)B_alpha) = nnn_(alpha in I)(A uu B_alpha)`,
    `(uuu_(alpha in I) A_alpha) nn (uuu_(beta in J) B_beta) = uuu_(alpha in I, beta in J) A_alpha nn B_beta`,
    `(nnn_(alpha in I) A_alpha) uu (nnn_(beta in J) B_beta) = nnn_(alpha in I, beta in J) A_alpha uu B_beta`, 但 `uuu_(alpha in I) nnn_(beta in J) A_(alpha, beta) = nnn_(beta in J) uuu_(alpha in I) A_(alpha, beta)` 一般不成立.
  1. 差: `B \\ A = B - A := {x: x in B and x !in A}`.
  2. 补 (余): 设 `X` 为全集, `A^c = bar A = ∁ A := X \\ A = {x: x !in A}`.

`(A^c)^c = A`, `quad` `A \\ B = A nn B^c`, `quad` `A = (A nn B) uu (A \\ B)`,
`A sube B rArr B^c sube A^c`, `quad` `A nn B = O/ iff A sube B^c`, `quad` `A \\ B = O/ iff A sube B`.

De. Morgan 法则 `B\\(uuu_(alpha in I) A_alpha) = nnn_(alpha in I) (B\\A_alpha)`, `quad` `B\\(nnn_(alpha in I) A_alpha) = uuu_(alpha in I) (B\\A_alpha)`;
`(uuu_(alpha in I) A_alpha)^c = nnn_(alpha in I) A_alpha^c`,`quad` `(nnn_(alpha in I) A_alpha)^c = uuu_(alpha in I) A_alpha^c`.

对称差 `A triangle B := (A \\ B) uu (B \\ A) = (A uu B) \\ (A nn B)`.

`(EE C) A triangle C = B triangle C iff A = B`,
`(EE C) A nn C = B nn C and A uu C = B uu C iff A = B`.

基数 (势): `|X|`. 对有限集合而言, 基数就是集合中元素的个数. 无限集的基数将在后面讨论.

幂集: 由 `X` 的一切子集形成的集合 `2^X = cc P(X) := {A: A sube X}`. 即 `A in 2^X iff A sube X`. 采用 `2^X` 这样的记号是因为, 若 `|X| = n`, 则 `|2^X| = 2^n`. 注意区分幂集与集合族的并这两个概念.
把 `2^X` 的子集称为 `X` 的子集族. 如果 `tau` 是 `X` 的子集族, 那么 `tau sube 2^X`, 即 `tau in 2^(2^X)`.

Descartes 积, 二元关系

Descartes 积 `A xx B := {(a, b): a in A, b in B}`, 其中 `(a, b)` 为有序二元组. ` prod_(i=1)^n X_i := {(x_1, x_2, cdots, x_n): x_i in X_i, i = 1, 2, cdots, n}`, 其中 `(x_1, x_2, cdots, x_n)` 为有序 `n` 元组. `X^n = prod_(i=1)^n X`.

集合 `X` 上的二元关系是 `X xx X` 的子集. 记 `X` 上的全体二元关系为 `cc B(X) := 2^(X xx X)`, 则 `sigma in cc B(X) iff sigma sube X xx X`. 设 `sigma` 是 `X` 上的一个二元关系, 称 `a` 与 `b` 是`sigma`-相关的, 如果 `(a, b) in sigma`, 记为 `a sigma b`: `a sigma b iff (a, b) in sigma`.

令 `X` 为非空集, 定义 `X` 上 (通常的) 二元关系合成 `sigma @ tau := {(a, b): (EE c in X)\ a sigma c and c tau b}`, `sigma, tau in cc B(X)`. 换言之, `a, b` 有关系 `sigma @ tau` 当且仅当存在中间元 `c in X`, `a sigma c` 且 `c tau b`.
